
Generate a Modelica model named `Add` that adds two integer input parameters 'u1' and 'u2' and outputs their sum 'y'. The model should extend the `IndustrialControlSystems.MathOperations.IntegerType.Interfaces.IntSimpleOperation`. The input 'u' and output 'y' exist in IntSimpleOperation and does not need to be redeclared. It should handle the addition through a conditional operation based on the sampling time Ts. If Ts is greater than zero, the output is computed at each sample point; otherwise, the output is the direct sum of the two inputs.

//modelica
```model Add
extends IndustrialControlSystems.MathOperations.IntegerType.Interfaces.IntSimpleOperation;
parameter IntegerType.Interfaces.IntSimpleOperation.u1;
parameter IntegerType.Interfaces.IntSimpleOperation.u2;
IntegerType.Interfaces.IntSimpleOperation.y = if Ts > 0 then u1 + u2 else u1 + u2;
end Add;
```
