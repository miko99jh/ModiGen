```modelica
model StepVoltage
  extends ModelicaByExample.Components.Electrical.DryApproach.TwoPin;

  parameter Modelica.SIunits.Voltage V0 = 0 "Initial voltage";
  parameter Modelica.SIunits.Voltage Vf = 1 "Final voltage";
  parameter Modelica.SIunits.Time stepTime = 1 "Time at which voltage steps from V0 to Vf";

  Modelica.SIunits.Voltage v;

equation
  // Voltage output switches from V0 to Vf at stepTime
  v = if time < stepTime then V0 else Vf;

  // Connect the voltage to the TwoPin interface
  p.v = v;
  n.v = 0; // Assuming the negative pin is grounded
end StepVoltage;
```

This Modelica model defines a step voltage source that extends the `TwoPin` class from the `ModelicaByExample.Components.Electrical.DryApproach` package. The model includes three parameters: `V0`, `Vf`, and `stepTime`, which control the initial and final voltages and the time at which the voltage steps from `V0` to `Vf`. The voltage `v` is calculated based on the current simulation time and is connected to the positive pin `p.v`, while the negative pin `n.v` is assumed to be grounded.
