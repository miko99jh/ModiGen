```modelica
model ZDiode
  extends Modelica.Electrical.Analog.Interfaces.OnePort;
  extends Modelica.Electrical.Analog.Interfaces.ConditionalHeatPort(T);

  parameter Modelica.Units.SI.Current Ids = 1e-6 "Saturation current";
  parameter Modelica.Units.SI.Voltage Vt = 0.04 "Voltage equivalent of temperature (kT/qn)";
  parameter Real Maxexp(final min=Modelica.Constants.small) = 30 "Maximum exponent limit";
  parameter Modelica.Units.SI.Resistance R = 1e8 "High resistance for reverse bias";
  parameter Modelica.Units.SI.Voltage Bv = 5.1 "Zener breakdown voltage";
  parameter Modelica.Units.SI.Current Ibv = 0.7 "Current at breakdown voltage";
  parameter Real Nbv = 0.74 "Emission coefficient for breakdown region";
  parameter Real T = 293.15 "Default temperature in Kelvin";

  Modelica.Units.SI.Current i "Current through the diode";
  Modelica.Units.SI.Voltage v "Voltage across the diode";
  Modelica.Units.SI.Power LossPower "Power loss in the diode";

equation
  v = port.v;
  i = smooth(1, if (v > Maxexp * Vt) then
        Ids * (exp(Maxexp) * (1 + v / Vt - Maxexp) - 1) + v / R
      else if ((v + Bv) < -Maxexp * (Nbv * Vt)) then
        -Ids - Ibv * exp(Maxexp) * (1 - (v + Bv) / (Nbv * Vt) - Maxexp) + v / R
      else
        Ids * (exp(v / Vt) - 1) - Ibv * exp(-(v + Bv) / (Nbv * Vt)) + v / R);

  LossPower = v * i;

  port.i = i;
end ZDiode;
```

This Modelica model defines a Zener diode with three distinct operating regions: forward bias, reverse bias, and breakdown (Zener) region. The model uses a piecewise function to calculate the current through the diode based on the voltage across it. The power loss is calculated as the product of voltage and current. The model also includes a conditional heat port with a default temperature value.
