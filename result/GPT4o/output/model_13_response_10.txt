```modelica
model Xnor
  extends IndustrialControlSystems.Logical.LogicalOperations.Interfaces.LogicalBlock(
    nInput=2, final nOutput=1);

  parameter Real Ts = 0 "Sample time";

equation
  // Assert that the number of inputs is at least 2
  assert(nInput >= 2, "The number of inputs must be at least 2.");

  if Ts > 0 then
    // Sample the input signals at regular intervals if Ts > 0
    when sample(0, Ts) then
      y[1] = not Functions.Xor(u[:], nInput);
    end when;
  else
    // Directly perform the XNOR operation if Ts <= 0
    y[1] = not Functions.Xor(u[:], nInput);
  end if;
end Xnor;
```

This Modelica model defines an `Xnor` gate that extends a logical block interface with two inputs and one output. It checks that the number of inputs is at least two and performs an XNOR operation on the inputs. If the sample time `Ts` is greater than zero, the operation is performed at regular intervals; otherwise, it is executed directly.
