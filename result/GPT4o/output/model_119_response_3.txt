```modelica
model LimitedActuator
  extends ModelicaByExample.Architectures.SensorComparison.Interfaces.Actuator;

  parameter Modelica.SIunits.Time delayTime;
  parameter Real uMax;

protected 
  Modelica.Mechanics.Rotational.Sources.Torque torque(useSupport=true);
  Modelica.Blocks.Nonlinear.Limiter limiter(uMax=uMax);
  Modelica.Blocks.Nonlinear.FixedDelay lag(delayTime=delayTime);

equation 
  // Mechanical Output
  connect(torque.flange, shaft);
  connect(torque.support, housing);

  // Signal Chain
  connect(tau, lag.u);
  connect(lag.y, limiter.u);

  // Actuation
  connect(limiter.y, torque.tau);

end LimitedActuator;
```

This Modelica model defines a `LimitedActuator` that extends a base actuator interface and incorporates a delay and saturation mechanism. The model uses a torque source, a limiter, and a fixed delay to process the input torque command, ensuring that the output is both delayed and limited according to the specified parameters.
