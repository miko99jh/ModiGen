```modelica
model Gain
  extends Modelica.Electrical.Analog.Examples.OpAmps.OpAmpCircuits.PartialOpAmp;

  parameter Real k(min=0) = 1 "Amplification factor";
  parameter Modelica.Units.SI.Resistance R1 = 1000 "Resistance at the inverting input";
  parameter Modelica.Units.SI.Resistance R2 = k * R1 "Feedback resistance";

  Modelica.Electrical.Analog.Basic.Resistor r1(R=R1) "Input resistor";
  Modelica.Electrical.Analog.Basic.Resistor r2(R=R2) "Feedback resistor";

equation
  // Inverting Path
  connect(p1, r1.p);
  connect(r1.n, in_n);

  // Feedback Loop
  connect(out, r2.p);
  connect(r2.n, in_n);

  // Common Reference
  connect(in_p, n1);
  connect(n1, n2);

  // Output Stage
  connect(out, p2);

end Gain;
```

This Modelica model defines an inverting operational amplifier circuit with a user-defined gain. The model inherits from a partial operational amplifier class and includes resistors to set the gain and feedback paths. The connections are established to simulate the behavior of an inverting amplifier accurately.
