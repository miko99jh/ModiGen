```modelica
model ClassicModel
  // Parameters
  parameter Real alpha = 0.1 "Reproduction rate of prey";
  parameter Real beta = 0.02 "Mortality rate of prey per predator";
  parameter Real gamma = 0.4 "Natural mortality rate of predator";
  parameter Real delta = 0.02 "Reproduction rate of predator per prey";

  // State variables
  Real x(start=10) "Prey population";
  Real y(start=10) "Predator population";

equation
  // Differential equations governing the dynamics
  der(x) = alpha * x - beta * x * y;
  der(y) = delta * x * y - gamma * y;
end ClassicModel;
```

This Modelica model, `ClassicModel`, simulates the dynamics of a predator-prey system using the specified parameters and initial conditions. The equations reflect the interactions between prey and predators, capturing the essence of population dynamics in an ecosystem.
