Certainly! Below is a Modelica model named `NewtonCooling` that implements Newton's law of cooling. This model uses the given parameters and equations to simulate the cooling process over time.

```modelica
model NewtonCooling
  // Parameters
  parameter Real T_inf = 300.0 "Ambient temperature (K)";
  parameter Real T0 = 280.0 "Initial temperature (K)";
  parameter Real h = 0.7 "Convective cooling coefficient (W/m^2.K)";
  parameter Real A = 1.0 "Surface area (m^2)";
  parameter Real m = 0.1 "Mass of thermal capacitance (kg)";
  parameter Real c_p = 1.2 "Specific heat capacity (J/kg.K)";

  // Variables
  Real T(start = T0) "Temperature of the object (K)";

equation
  // Newton's law of cooling
  m * c_p * der(T) = h * A * (T_inf - T);

end NewtonCooling;
```

### Explanation:
- **Parameters**: These are the constants used in the model:
  - `T_inf`: Ambient temperature.
  - `T0`: Initial temperature of the object.
  - `h`: Convective cooling coefficient.
  - `A`: Surface area through which heat is transferred.
  - `m`: Mass of the object.
  - `c_p`: Specific heat capacity of the object.

- **Variables**:
  - `T`: The temperature of the object, which changes over time. It is initialized to `T0`.

- **Equation**:
  - The equation `m * c_p * der(T) = h * A * (T_inf - T)` represents Newton's law of cooling, where `der(T)` is the time derivative of temperature, indicating how the temperature changes over time.

This model can be simulated to observe how the temperature of the object approaches the ambient temperature over time.
