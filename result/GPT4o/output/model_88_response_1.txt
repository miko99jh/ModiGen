To implement the Modelica model `RLC1` for the described RLC circuit, you can follow the structure below. This model will simulate the behavior of the circuit with the given parameters and variables.

```modelica
model RLC1
  // Parameters
  parameter Real R = 100;       // Resistance in ohms
  parameter Real C = 1e-3;      // Capacitance in farads
  parameter Real L = 1;         // Inductance in henrys
  parameter Real Vb = 24;       // Supply voltage in volts

  // Variables
  Real V;                       // Voltage across the parallel branch
  Real i_L;                     // Current through the inductor
  Real i_R;                     // Current through the resistor
  Real i_C;                     // Current through the capacitor

  // Equations
equation
  // Kirchhoff's voltage law for the series connection
  Vb = V + L * der(i_L);

  // Kirchhoff's current law for the parallel branch
  i_L = i_R + i_C;

  // Ohm's law for the resistor
  V = R * i_R;

  // Capacitor equation
  i_C = C * der(V);

  // Initial conditions (optional, can be adjusted as needed)
  initial equation
    V = 0;
    i_L = 0;
    i_R = 0;
    i_C = 0;
end RLC1
```

### Explanation:
- **Parameters**: These are the fixed values for resistance, capacitance, inductance, and supply voltage.
- **Variables**: These represent the dynamic quantities in the circuit, such as voltages and currents.
- **Equations**:
  - Kirchhoff's voltage law is applied to the series connection of the inductor and the parallel branch.
  - Kirchhoff's current law is applied to the parallel branch, ensuring the sum of currents through the resistor and capacitor equals the current through the inductor.
  - Ohm's law is used to relate the voltage across the resistor to the current through it.
  - The capacitor equation relates the current through the capacitor to the rate of change of voltage across it.
- **Initial Conditions**: These are optional and can be adjusted based on the specific simulation requirements.

This model can be simulated using a Modelica environment to observe the transient and steady-state behavior of the RLC circuit.
