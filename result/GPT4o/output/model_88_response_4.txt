To implement the `RLC1` Modelica model for the described RLC circuit, you can follow the structure below. This model will simulate the behavior of the circuit with the given parameters and variables.

```modelica
model RLC1
  // Parameters
  parameter Real R = 100;      // Resistance in ohms
  parameter Real C = 1e-3;     // Capacitance in farads
  parameter Real L = 1;        // Inductance in henrys
  parameter Real Vb = 24;      // Supply voltage in volts

  // Variables
  Real V;                      // Voltage across the parallel branch
  Real i_L;                    // Current through the inductor
  Real i_R;                    // Current through the resistor
  Real i_C;                    // Current through the capacitor

  // Equations
equation
  // Kirchhoff's voltage law for the series circuit
  Vb = V + L * der(i_L);

  // Kirchhoff's current law for the parallel branch
  i_L = i_R + i_C;

  // Ohm's law for the resistor
  V = R * i_R;

  // Capacitor current-voltage relationship
  i_C = C * der(V);

  // Initial conditions (optional, depending on simulation needs)
  // V = 0;
  // i_L = 0;

end RLC1;
```

### Explanation:
- **Parameters**: The model defines the resistance `R`, capacitance `C`, inductance `L`, and supply voltage `Vb` as parameters.
- **Variables**: The model uses `V` for the voltage across the parallel branch, `i_L` for the current through the inductor, `i_R` for the current through the resistor, and `i_C` for the current through the capacitor.
- **Equations**:
  - Kirchhoff's voltage law is applied to the series circuit, which includes the inductor and the parallel branch.
  - Kirchhoff's current law is applied to the parallel branch, ensuring the sum of currents through the resistor and capacitor equals the current through the inductor.
  - Ohm's law is used to relate the voltage across the resistor to the current through it.
  - The capacitor's current-voltage relationship is defined using its capacitance and the derivative of the voltage across it.
