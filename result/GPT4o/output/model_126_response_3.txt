Certainly! Below is a simple Modelica model named `FirstOrderExperiment` that defines a first-order differential equation with experiment conditions. The model includes a real variable `x`, initialized to 2, and aims for `x` to converge towards 1.0.

```modelica
model FirstOrderExperiment
  // Define a real variable 'x' initialized to 2
  Real x(start=2);

equation
  // Define the first-order differential equation
  der(x) = -0.5 * (x - 1);

  // The equation above represents a simple first-order system
  // where 'x' converges towards 1.0 over time.

annotation(
  experiment(StartTime=0, StopTime=10, Tolerance=1e-6, Interval=0.1)
);

end FirstOrderExperiment;
```

### Explanation:
- **Variable Initialization**: The real variable `x` is initialized to 2 using `start=2`.
- **Differential Equation**: The equation `der(x) = -0.5 * (x - 1)` describes how `x` changes over time. The term `-0.5 * (x - 1)` represents a proportional control that drives `x` towards 1.0.
- **Experiment Annotation**: The `annotation(experiment(...))` specifies the simulation conditions:
  - `StartTime=0`: The simulation starts at time 0.
  - `StopTime=10`: The simulation runs until time 10.
  - `Tolerance=1e-6`: The solver tolerance for numerical integration.
  - `Interval=0.1`: The output interval for the simulation results.

This model will simulate how `x` changes over time, starting from 2 and converging towards 1.0 due to the defined differential equation.
