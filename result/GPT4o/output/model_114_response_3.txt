```modelica
model SecondOrderSystem
  import Modelica.SIunits.*;

  // Parameters
  parameter Angle phi1_init = 0;
  parameter Angle phi2_init = 1;
  parameter AngularVelocity omega1_init = 0;
  parameter AngularVelocity omega2_init = 0;
  parameter Inertia J1 = 0.4;
  parameter Inertia J2 = 1.0;
  parameter RotationalSpringConstant c1 = 11;
  parameter RotationalSpringConstant c2 = 5;
  parameter RotationalDampingConstant d1 = 0.2;
  parameter RotationalDampingConstant d2 = 1.0;

  // Variables
  Angle phi1;
  Angle phi2;
  AngularVelocity omega1;
  AngularVelocity omega2;

initial equation
  phi1 = phi1_init;
  phi2 = phi2_init;
  omega1 = omega1_init;
  omega2 = omega2_init;

equation
  // Kinematics
  der(phi1) = omega1;
  der(phi2) = omega2;

  // Kinetics for J1
  J1 * der(omega1) = -c1 * (phi1 - phi2) - d1 * (omega1 - omega2);

  // Kinetics for J2
  J2 * der(omega2) = c1 * (phi1 - phi2) + d1 * (omega1 - omega2) - c2 * phi2 - d2 * omega2;
end SecondOrderSystem
```

This Modelica model represents a second-order rotational system with two inertias coupled by a spring and damper, and includes initial conditions for angles and angular velocities. The equations describe the kinematics and kinetics of the system, including the coupling between the two inertias.
