To simulate the flow of fluid through a pipe using Modelica, you can create a model named `Pipe` that uses the Hagen-Poiseuille relationship. This relationship is applicable for laminar flow in a cylindrical pipe and can be used to calculate the flow rate (`Q`), flow coefficient (`c`), and velocity (`v`). Below is a Modelica model that implements this:

```modelica
model Pipe
  // Parameters
  parameter Real P1 = 1080; // Pressure at the inlet [Pa]
  parameter Real P2 = 900;  // Pressure at the outlet [Pa]
  parameter Real L = 1;     // Length of the pipe [m]
  parameter Real D = 0.05;  // Inner diameter of the pipe [m]
  parameter Real mu = 2e-3; // Viscosity of the fluid [Pa.s]

  // Variables
  Real c;  // Flow coefficient
  Real Q;  // Flow rate [m^3/s]
  Real v;  // Velocity [m/s]
  Real deltaP; // Pressure difference [Pa]
  Real A;  // Cross-sectional area [m^2]

equation
  // Calculate pressure difference
  deltaP = P1 - P2;

  // Calculate cross-sectional area
  A = (pi * D^2) / 4;

  // Calculate flow coefficient using Hagen-Poiseuille equation
  c = (pi * D^4) / (128 * mu * L);

  // Calculate flow rate
  Q = c * deltaP;

  // Calculate velocity
  v = Q / A;
end Pipe;
```

### Explanation:
- **Parameters**: These are the known values for pressure (`P1`, `P2`), pipe length (`L`), diameter (`D`), and fluid viscosity (`mu`).
- **Variables**: These are the calculated values for flow coefficient (`c`), flow rate (`Q`), velocity (`v`), pressure difference (`deltaP`), and cross-sectional area (`A`).
- **Equations**:
  - `deltaP` is the pressure difference between the two ends of the pipe.
  - `A` is the cross-sectional area of the pipe.
  - `c` is calculated using the Hagen-Poiseuille equation, which relates the flow rate to the pressure difference, pipe dimensions, and fluid viscosity.
  - `Q` is the flow rate, calculated using the flow coefficient and pressure difference.
  - `v` is the velocity of the fluid, calculated from the flow rate and cross-sectional area.

This model assumes laminar flow and is based on the Hagen-Poiseuille equation, which is valid for incompressible, Newtonian fluids in cylindrical pipes.
