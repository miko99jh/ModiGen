The model then calculates the absolute values of the array elements, the minimum value, the maximum value, and the sum of all elements using an equation block.

```modelica
model ArrayExample
  // Define the array and its length
  parameter Integer n = 5;
  parameter Integer data[n] = {1, 2, 3, 4, 5};

  // Variables to store results
  Integer abs1[n];
  Integer min1;
  Integer max1;
  Integer sum1;

equation
  // Calculate absolute values
  for i in 1:n loop
    abs1[i] = abs(data[i]);
  end for;

  // Calculate minimum value
  min1 = min(data);

  // Calculate maximum value
  max1 = max(data);

  // Calculate sum of all elements
  sum1 = sum(data);

end ArrayExample;
```

### Explanation:
- **Array Definition**: The array `data` is defined with a length of `n = 5` and initialized with the values `{1, 2, 3, 4, 5}`.
- **Absolute Values**: The `abs1` array stores the absolute values of each element in `data`. Since all elements are positive, `abs1` will be the same as `data`.
- **Minimum Value**: The `min1` variable stores the minimum value of the array, which is calculated using the `min()` function.
- **Maximum Value**: The `max1` variable stores the maximum value of the array, calculated using the `max()` function.
- **Sum of Elements**: The `sum1` variable stores the sum of all elements in the array, calculated using the `sum()` function.

This model uses an equation block to perform the calculations, ensuring that the results are computed based on the initialized values of the array.
