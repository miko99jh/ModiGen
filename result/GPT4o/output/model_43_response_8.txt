```modelica
block Add
  extends Modelica.ComplexBlocks.Interfaces.ComplexSI2SO;

  parameter Modelica.ComplexBlocks.Complex k1 = Complex(1, 0) "Gain factor for input u1Internal";
  parameter Modelica.ComplexBlocks.Complex k2 = Complex(1, 0) "Gain factor for input u2Internal";

protected
  Modelica.ComplexBlocks.Complex u1Weighted;
  Modelica.ComplexBlocks.Complex u2Weighted;

equation
  // Compute weighted inputs
  u1Weighted = k1 * u1Internal;
  u2Weighted = k2 * u2Internal;

  // Output the sum of weighted inputs
  y = u1Weighted + u2Weighted;

end Add;
```

This Modelica block named `Add` computes the sum of two complex input signals, each multiplied by a respective gain factor (`k1` and `k2`). The block extends from `Modelica.ComplexBlocks.Interfaces.ComplexSI2SO`, ensuring it follows the specified two-input, single-output interface with complex signal types. The parameters `k1` and `k2` are complex numbers with default values of `Complex(1, 0)`. The block calculates the weighted inputs and outputs their sum as a complex signal `y`.
