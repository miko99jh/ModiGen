```modelica
model ClassicModel
  // Parameters
  parameter Real alpha = 0.1 "Reproduction rate of prey";
  parameter Real beta = 0.02 "Mortality rate of predator per prey";
  parameter Real gamma = 0.4 "General mortality rate of predator";
  parameter Real delta = 0.02 "Reproduction rate of predator per prey";

  // State variables
  Real x(start=10) "Prey population";
  Real y(start=10) "Predator population";

equation
  // Differential equations
  der(x) = alpha * x - beta * x * y;
  der(y) = delta * x * y - gamma * y;
end ClassicModel;
```

### Explanation:
- **Parameters**: These are constants that define the rates of reproduction and mortality for both prey and predators.
  - `alpha`: Reproduction rate of prey.
  - `beta`: Mortality rate of predator per prey.
  - `gamma`: General mortality rate of predator.
  - `delta`: Reproduction rate of predator per prey.

- **State Variables**: These represent the populations of prey (`x`) and predators (`y`), with initial values set to 10 for both.

- **Equations**:
  - `der(x) = alpha * x - beta * x * y`: The rate of change of the prey population is determined by its reproduction rate minus the rate of predation.
  - `der(y) = delta * x * y - gamma * y`: The rate of change of the predator population is determined by the rate of reproduction from consuming prey minus its natural mortality rate.

This model can be used to simulate and analyze the dynamics of predator-prey interactions in an ecosystem.
