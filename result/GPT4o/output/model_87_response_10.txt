Certainly! Below is a simple Modelica model for simulating a single pendulum. This model uses the basic physics of a pendulum, where the pendulum is positioned on a circle with a radius \( L = 1 \) meter, and the mass of the pendulum is \( m = 1 \) kg. The gravitational constant \( g \) is set to \( 9.81 \, \text{m/s}^2 \). The initial angle \( \theta \) is set to \( 1.5 \) radians, and the initial angular velocity \( \omega \) is set to \( 0 \) rad/s.

```modelica
model SimplePendulum
  parameter Real L = 1; // Length of the pendulum
  parameter Real m = 1; // Mass of the pendulum
  parameter Real g = 9.81; // Gravitational acceleration

  // State variables
  Real theta(start=1.5); // Angle from the vertical in radians
  Real omega(start=0); // Angular velocity in rad/s

  // Differential equation for the pendulum motion
equation
  der(theta) = omega;
  der(omega) = -(g/L) * sin(theta);

  // Output for visualization
  Real x = L * sin(theta);
  Real y = -L * cos(theta);
end SimplePendulum;
```

### Explanation:
- **Parameters**: 
  - `L`: Length of the pendulum (1 meter).
  - `m`: Mass of the pendulum (1 kg).
  - `g`: Gravitational acceleration (9.81 m/s²).

- **State Variables**:
  - `theta`: Angle from the vertical in radians, initialized to 1.5 radians.
  - `omega`: Angular velocity in rad/s, initialized to 0 rad/s.

- **Equations**:
  - `der(theta) = omega`: The derivative of the angle is the angular velocity.
  - `der(omega) = -(g/L) * sin(theta)`: The derivative of the angular velocity is determined by the gravitational force acting on the pendulum.

- **Output**:
  - `x` and `y`: These are calculated for visualization purposes, representing the Cartesian coordinates of the pendulum bob.

This model can be simulated using a Modelica environment like OpenModelica or Dymola to observe the pendulum's motion over time.
