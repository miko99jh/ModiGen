```modelica
model IdealGTOThyristor
  extends Modelica.Electrical.Analog.Interfaces.IdealSemiconductor;

  Modelica.Blocks.Interfaces.BooleanInput fire "Boolean input for firing the thyristor";

equation
  // The thyristor is off when the semiconductor state variable s is less than 0
  // or when the fire signal is false.
  off = (s < 0) or not fire;
end IdealGTOThyristor;
```

This Modelica model defines an ideal GTO thyristor by extending the `IdealSemiconductor` interface. The model includes a Boolean input `fire` to control the firing of the thyristor. The equation section specifies that the thyristor is off when the semiconductor state variable `s` is less than 0 or when the `fire` signal is false.
