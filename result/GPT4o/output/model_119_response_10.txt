```modelica
model LimitedActuator
  extends ModelicaByExample.Architectures.SensorComparison.Interfaces.Actuator;

  parameter Modelica.SIunits.Time delayTime;
  parameter Real uMax;

protected 
  Modelica.Mechanics.Rotational.Sources.Torque torque(useSupport=true);
  Modelica.Blocks.Nonlinear.Limiter limiter(uMax=uMax);
  Modelica.Blocks.Nonlinear.FixedDelay lag(delayTime=delayTime);

equation 
  // Mechanical Output
  connect(torque.flange, shaft);
  connect(torque.support, housing);

  // Signal Chain
  connect(tau, lag.u);
  connect(lag.y, limiter.u);

  // Actuation
  connect(limiter.y, torque.tau);

end LimitedActuator;
```

This Modelica model defines a `LimitedActuator` that extends the specified interface and incorporates components for delay and saturation. The model ensures that the input torque command is processed through a delay and limiter before driving the torque source, with appropriate connections to the mechanical output.
