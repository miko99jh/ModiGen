To implement a `BouncingBall` model in Modelica, you can follow the structure below. This model simulates a ball bouncing under the influence of gravity, with a specified coefficient of restitution for the bounces.

```modelica
model BouncingBall
  // Parameters
  parameter Real radius = 0.1 "Radius of the ball in meters";
  parameter Real e = 0.9 "Coefficient of restitution";
  parameter Real g = 9.81 "Acceleration due to gravity in m/s^2";

  // State variables
  Real h(start = 1) "Height of the ball in meters";
  Real v(start = 0) "Velocity of the ball in m/s";

  // Event to detect collision with the ground
  Boolean isOnGround;

equation
  // Motion equations
  der(h) = v;
  der(v) = if isOnGround then 0 else -g;

  // Collision detection
  isOnGround = h <= radius and v <= 0;

  // Collision response
  when isOnGround then
    v = -e * v;
  end when;

  // Ensure the ball does not go below the ground
  assert(h >= radius, "Ball has penetrated the ground!");

end BouncingBall;
```

### Explanation:
- **Parameters**: 
  - `radius`: The radius of the ball.
  - `e`: The coefficient of restitution, which determines how much velocity is retained after a bounce.
  - `g`: The gravitational acceleration.

- **State Variables**:
  - `h`: The height of the ball.
  - `v`: The velocity of the ball.

- **Equations**:
  - The equations of motion are defined using `der(h)` and `der(v)`, representing the derivatives of height and velocity, respectively.
  - The `isOnGround` boolean variable is used to detect when the ball is in contact with the ground.
  - The `when` statement is used to handle the collision response, reversing the velocity and applying the coefficient of restitution.

- **Assertions**:
  - An assertion is included to ensure the ball does not penetrate the ground, which helps in debugging and ensuring the physical validity of the simulation.

This model will simulate the ball bouncing under the influence of gravity, with the specified initial conditions and parameters. You can simulate this model using a Modelica simulation environment like OpenModelica or Dymola.
