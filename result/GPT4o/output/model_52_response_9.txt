To create a Modelica model named `ShowSaturatingInductor` that demonstrates the behavior of a `SaturatingInductor` component, you can follow the structure below. This model includes the specified parameters and components, and sets up the connections as described:

```modelica
model ShowSaturatingInductor
  // Parameters
  parameter Modelica.Units.SI.Inductance Lzer = 2 "Inductance near zero current";
  parameter Modelica.Units.SI.Inductance Lnom = 1 "Nominal inductance at nominal current";
  parameter Modelica.Units.SI.Current Inom = 1 "Nominal current";
  parameter Modelica.Units.SI.Inductance Linf = 0.5 "Inductance at large currents";
  parameter Modelica.Units.SI.Voltage U = 1.25 "Source voltage peak";
  parameter Modelica.Units.SI.Frequency f = 1/(2*Modelica.Constants.pi) "Source frequency";
  parameter Modelica.Units.SI.Angle phase = Modelica.Constants.pi/2 "Phase shift for source voltage";

  // Components
  Modelica.Electrical.Analog.Sources.SineVoltage SineVoltage1(V=U, phase=phase, f=f) "Sine voltage source";
  Modelica.Electrical.Analog.Basic.Ground Ground1 "Ground component";
  Modelica.Electrical.Analog.Basic.SaturatingInductor SaturatingInductance1(Lzer=Lzer, Lnom=Lnom, Inom=Inom, Linf=Linf, i(fixed=true)) "Saturating inductor";
  Modelica.Electrical.Analog.Basic.Inductor Inductance1(L=Lnom, i(fixed=true)) "Standard inductor";

  // Connections
  equation
    connect(SineVoltage1.n, Ground1.p); // Ground the negative terminal of the sine voltage source
    connect(SineVoltage1.p, SaturatingInductance1.p); // Connect positive terminal of sine voltage to saturating inductor
    connect(SineVoltage1.p, Inductance1.p); // Connect positive terminal of sine voltage to standard inductor
    connect(SaturatingInductance1.n, SineVoltage1.n); // Connect negative terminal of saturating inductor to negative terminal of sine voltage
    connect(Inductance1.n, SineVoltage1.n); // Connect negative terminal of standard inductor to negative terminal of sine voltage
end ShowSaturatingInductor
```

This model sets up a simple circuit with a sine voltage source driving two parallel branches: one with a saturating inductor and the other with a standard inductor. The parameters are defined as specified, and the connections ensure that both inductors are subjected to the same voltage source, allowing you to observe the behavior of the saturating inductor compared to a standard inductor.
