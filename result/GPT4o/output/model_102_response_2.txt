To create a Modelica model named `ArrayExample` that defines an array and calculates the absolute values, minimum, maximum, and sum of its elements, you can use the following code. This code uses an equation block to perform the necessary calculations:

```modelica
model ArrayExample
  // Define the array with initial values
  parameter Integer n = 5;
  Integer data[n] = {1, 2, 3, 4, 5};
  
  // Variables to store results
  Integer abs1[n];
  Integer min1;
  Integer max1;
  Integer sum1;

equation
  // Calculate absolute values
  for i in 1:n loop
    abs1[i] = abs(data[i]);
  end for;

  // Calculate minimum value
  min1 = min(data);

  // Calculate maximum value
  max1 = max(data);

  // Calculate sum of all elements
  sum1 = sum(data);

end ArrayExample;
```

### Explanation:
- **Array Definition**: The array `data` is defined with a length of `n = 5` and initialized with the values `{1, 2, 3, 4, 5}`.
- **Absolute Values**: The `abs1` array is calculated using a `for` loop that iterates over each element of `data` and applies the `abs` function.
- **Minimum and Maximum Values**: The `min1` and `max1` are calculated using the `min` and `max` functions, respectively, which operate on the entire array.
- **Sum of Elements**: The `sum1` is calculated using the `sum` function, which adds up all the elements of the array.

This Modelica model will compute the desired values when simulated.
