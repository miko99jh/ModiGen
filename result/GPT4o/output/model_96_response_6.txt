Below is the Modelica model named `Pipe` that simulates the flow of fluid through a pipe using the Hagen-Poiseuille relationship. The model calculates the flow coefficient (`c`), the flow rate (`Q`), and the velocity (`v`).

```modelica
model Pipe
  // Parameters
  parameter Real P1 = 1080; // Pressure at the inlet (Pa)
  parameter Real P2 = 900;  // Pressure at the outlet (Pa)
  parameter Real L = 1;     // Length of the pipe (m)
  parameter Real D = 0.05;  // Inner diameter of the pipe (m)
  parameter Real mu = 2e-3; // Viscosity of the fluid (Pa·s)

  // Variables
  Real c;                   // Flow coefficient (m^3/s/Pa)
  Real Q;                   // Flow rate (m^3/s)
  Real v;                   // Velocity (m/s)
  Real deltaP;              // Pressure difference (Pa)
  Real A;                   // Cross-sectional area of the pipe (m^2)

equation
  // Calculate pressure difference
  deltaP = P1 - P2;

  // Cross-sectional area of the pipe
  A = pi * (D/2)^2;

  // Hagen-Poiseuille flow coefficient
  c = (pi * D^4) / (128 * mu * L);

  // Flow rate using Hagen-Poiseuille relationship
  Q = c * deltaP;

  // Velocity derived from flow rate and cross-sectional area
  v = Q / A;
end Pipe;
```

### Explanation:
1. **Parameters**:
   - `P1` and `P2` are the pressures at the inlet and outlet of the pipe, respectively.
   - `L` is the length of the pipe.
   - `D` is the inner diameter of the pipe.
   - `mu` is the dynamic viscosity of the fluid.

2. **Variables**:
   - `c` is the flow coefficient, which depends on the pipe dimensions and fluid viscosity.
   - `Q` is the volumetric flow rate through the pipe.
   - `v` is the fluid velocity in the pipe.
   - `deltaP` is the pressure difference between the inlet and outlet.
   - `A` is the cross-sectional area of the pipe.

3. **Equations**:
   - The pressure difference is calculated as `deltaP = P1 - P2`.
   - The cross-sectional area of the pipe is calculated using the formula for the area of a circle: `A = pi * (D/2)^2`.
   - The flow coefficient `c` is derived from the Hagen-Poiseuille equation: `c = (pi * D^4) / (128 * mu * L)`.
   - The flow rate `Q` is proportional to the pressure difference and the flow coefficient: `Q = c * deltaP`.
   - The velocity `v` is calculated by dividing the flow rate by the cross-sectional area: `v = Q / A`.

This model can be simulated in a Modelica environment to compute the values of `c`, `Q`, and `v` for the given parameters.
