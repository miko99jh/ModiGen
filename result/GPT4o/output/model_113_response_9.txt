To create a Modelica model for an RLC circuit using a state space representation, you can follow the structure below. This model uses the `ModelicaByExample.ArrayEquations.StateSpace.LTI` component to define the state space representation of the RLC circuit. The parameters and matrices are set according to your specifications.

```modelica
model RLC
  import ModelicaByExample.ArrayEquations.StateSpace.LTI;

  // Parameters
  parameter Real Vb = 24; // Voltage source in volts
  parameter Real L = 1;   // Inductance in Henry
  parameter Real R = 100; // Resistance in ohms
  parameter Real C = 0.001; // Capacitance in Farads

  // State space matrices
  parameter Real A[2, 2] = [-1/(R*C), 1/C; -1/L, 0];
  parameter Real B[2, 1] = [0; 1/L];
  parameter Real C[2, 2] = [1/R, 0; -1/R, 1];
  parameter Real D[2, 1] = [0; 0];

  // Initial state
  parameter Real x0[2] = {0, 0};

  // State space component
  LTI rlc_comp(
    nx = 2, 
    nu = 1, 
    ny = 2, 
    A = A, 
    B = B, 
    C = C, 
    D = D, 
    x0 = x0
  );

  // Voltage source
  Modelica.Blocks.Sources.Constant voltageSource(
    k = Vb
  );

equation
  // Connect the input 'u' of LTI component to the voltage source
  connect(voltageSource.y, rlc_comp.u[1]);

  // Outputs can be connected to other components or used for analysis
  // connect(rlc_comp.y, ...);

end RLC;
```

### Explanation:
- **Parameters**: The circuit parameters are defined as constants (`Vb`, `L`, `R`, `C`).
- **State Space Matrices**: The matrices `A`, `B`, `C`, and `D` are defined according to the state space representation of the RLC circuit.
- **Initial State**: The initial state `x0` is set to `{0, 0}`.
- **LTI Component**: The `LTI` component from `ModelicaByExample.ArrayEquations.StateSpace` is used to represent the state space model.
- **Voltage Source**: A constant voltage source is used to provide the input to the system.
- **Connections**: The voltage source output is connected to the input `u` of the `LTI` component.

This model provides a basic setup for simulating the behavior of an RLC circuit using state space representation in Modelica. You can further expand the model by connecting the outputs to other components or using them for analysis.
