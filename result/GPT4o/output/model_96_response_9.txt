To create a Modelica model named `Pipe` that simulates the flow of fluid through a pipe using the Hagen-Poiseuille relationship, you can follow the structure below. This model will calculate the flow coefficient (`c`), the flow in the pipeline (`Q`), and the velocity (`v`). The Hagen-Poiseuille equation for laminar flow in a circular pipe is used to determine these values.

```modelica
model Pipe
  // Parameters
  parameter Real P1 = 1080; // Pressure at the inlet (Pa)
  parameter Real P2 = 900;  // Pressure at the outlet (Pa)
  parameter Real L = 1;     // Length of the pipe (m)
  parameter Real D = 0.05;  // Inner diameter of the pipe (m)
  parameter Real mu = 2e-3; // Viscosity of the fluid (Pa.s)

  // Variables
  Real c; // Flow coefficient
  Real Q; // Flow rate (m^3/s)
  Real v; // Velocity (m/s)
  Real deltaP; // Pressure difference (Pa)
  Real A; // Cross-sectional area of the pipe (m^2)

equation
  // Calculate pressure difference
  deltaP = P1 - P2;

  // Calculate cross-sectional area
  A = pi * (D/2)^2;

  // Hagen-Poiseuille equation for flow coefficient
  c = (pi * D^4) / (128 * mu * L);

  // Calculate flow rate using Hagen-Poiseuille relationship
  Q = c * deltaP;

  // Calculate velocity
  v = Q / A;

end Pipe;
```

### Explanation:
- **Parameters**: These are the known values for pressure (`P1`, `P2`), pipe length (`L`), diameter (`D`), and fluid viscosity (`mu`).
- **Variables**: These are the calculated values for flow coefficient (`c`), flow rate (`Q`), velocity (`v`), pressure difference (`deltaP`), and cross-sectional area (`A`).
- **Equations**:
  - `deltaP` is the pressure difference between the two ends of the pipe.
  - `A` is the cross-sectional area of the pipe, calculated using the formula for the area of a circle.
  - `c` is the flow coefficient derived from the Hagen-Poiseuille equation.
  - `Q` is the flow rate, calculated using the flow coefficient and pressure difference.
  - `v` is the velocity of the fluid, calculated by dividing the flow rate by the cross-sectional area.

This model assumes laminar flow conditions, which is valid for small diameter pipes and low flow rates.
