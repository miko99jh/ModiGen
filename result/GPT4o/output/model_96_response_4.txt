To create a Modelica model for simulating fluid flow through a pipe using the Hagen-Poiseuille relationship, you can follow the structure below. This model calculates the flow coefficient (`c`), the flow rate (`Q`), and the velocity (`v`) based on the given parameters.

```modelica
model Pipe
  // Parameters
  parameter Real P1 = 1080; // Pressure at the inlet (Pa)
  parameter Real P2 = 900;  // Pressure at the outlet (Pa)
  parameter Real L = 1;     // Length of the pipe (m)
  parameter Real D = 0.05;  // Inner diameter of the pipe (m)
  parameter Real mu = 2e-3; // Viscosity of the fluid (Pa.s)

  // Calculated values
  Real c; // Flow coefficient
  Real Q; // Flow rate (m^3/s)
  Real v; // Velocity (m/s)

  // Constants
  constant Real pi = 3.141592653589793;

equation
  // Calculate the flow coefficient using the Hagen-Poiseuille equation
  c = (pi * D^4) / (128 * mu * L);

  // Calculate the flow rate using the pressure difference and flow coefficient
  Q = c * (P1 - P2);

  // Calculate the cross-sectional area of the pipe
  Real A = pi * (D/2)^2;

  // Calculate the velocity of the fluid
  v = Q / A;

end Pipe;
```

### Explanation:
- **Parameters**: These are the known values for pressure (`P1`, `P2`), pipe length (`L`), diameter (`D`), and fluid viscosity (`mu`).
- **Calculated Values**: These are the values we want to compute: flow coefficient (`c`), flow rate (`Q`), and velocity (`v`).
- **Constants**: The value of `pi` is used for calculations involving circular cross-sections.
- **Equations**:
  - The flow coefficient `c` is calculated using the Hagen-Poiseuille equation, which relates the flow rate to the pressure difference, pipe dimensions, and fluid viscosity.
  - The flow rate `Q` is determined by multiplying the flow coefficient `c` by the pressure difference (`P1 - P2`).
  - The cross-sectional area `A` of the pipe is calculated using the formula for the area of a circle.
  - The velocity `v` is derived by dividing the flow rate `Q` by the cross-sectional area `A`.

This model provides a basic simulation of fluid flow through a pipe using the Hagen-Poiseuille relationship, suitable for incompressible, laminar flow conditions.
