Certainly! Below is a simple Modelica model for a PID controller that regulates the water level in a container. The model includes proportional, integral, and differential control components to maintain the water level at the target height of 1 meter.

```modelica
model PIDController
  // Parameters
  parameter Real target_height = 1.0; // Target water level in meters
  parameter Real Kp = 1.0;            // Proportional coefficient
  parameter Real Ki = 0.2;            // Integral coefficient
  parameter Real Kd = 0.1;            // Differential coefficient
  parameter Real leak_rate = 0.1;     // Leak rate in meters per unit time

  // Variables
  Real waterLevel(start=0.2);         // Current water level in meters
  Real error;                         // Error in water level
  Real Pcontrol;                      // Proportional control
  Real Icontrol;                      // Integral control
  Real Dcontrol;                      // Differential control
  Real integralError(start=0.0);      // Accumulated error for integral control
  Real previousError(start=0.0);      // Previous error for differential control
  Real controlOutput;                 // Total control output

equation
  // Calculate error
  error = target_height - waterLevel;

  // Proportional control
  Pcontrol = Kp * error;

  // Integral control
  der(integralError) = error; // Accumulate error over time
  Icontrol = Ki * integralError;

  // Differential control
  Dcontrol = Kd * (error - previousError);
  previousError = error;

  // Total control output
  controlOutput = Pcontrol + Icontrol + Dcontrol;

  // Update water level
  der(waterLevel) = controlOutput - leak_rate;

end PIDController;
```

### Explanation:
- **Parameters**: These define the target height, PID coefficients, and leak rate.
- **Variables**: These include the current water level, error, and components of the PID control.
- **Equations**:
  - The error is calculated as the difference between the target height and the current water level.
  - The proportional control (`Pcontrol`) is directly proportional to the error.
  - The integral control (`Icontrol`) accumulates the error over time to address steady-state errors.
  - The differential control (`Dcontrol`) predicts future errors based on the rate of change of the error.
  - The water level is updated based on the control output minus the leak rate.

This model simulates the PID control system to maintain the water level at the desired height over time. You can simulate this model using a Modelica simulation environment to observe how the water level changes and stabilizes at the target height.
