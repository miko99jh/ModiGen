```modelica
function PolynomialWithDerivative
  input Real x; // Independent variable
  input Real c[:]; // Array of polynomial coefficients
  output Real y; // Computed polynomial value
  output Real dy; // Derivative of the polynomial
protected
  Integer n = size(c, 1); // Size of the coefficient array
algorithm
  y := 0.0; // Initialize polynomial value
  dy := 0.0; // Initialize derivative value
  for i in 1:n loop
    y := y + c[i] * x^(i-1); // Compute polynomial value
    if i > 1 then
      dy := dy + (i-1) * c[i] * x^(i-2); // Compute derivative value
    end if;
  end for;
end PolynomialWithDerivative;
```

### Explanation:
- **Inputs:**
  - `x`: The independent variable where the polynomial is evaluated.
  - `c`: An array of coefficients for the polynomial. The coefficient `c[i]` corresponds to the term `c[i] * x^(i-1)`.

- **Outputs:**
  - `y`: The value of the polynomial at `x`.
  - `dy`: The derivative of the polynomial at `x`.

- **Protected Variable:**
  - `n`: The number of coefficients, which determines the degree of the polynomial.

- **Algorithm:**
  - The function initializes `y` and `dy` to zero.
  - It iterates over each coefficient, updating the polynomial value `y` and its derivative `dy`.
  - The derivative is calculated using the power rule, where the derivative of `x^(i-1)` is `(i-1) * x^(i-2)`.

This function efficiently computes both the polynomial value and its derivative using a loop, making it suitable for various applications where polynomial evaluation and differentiation are required.
