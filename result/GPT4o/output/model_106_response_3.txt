To implement a Lorenz system in Modelica, you can define a model named `Lorenz` with the specified parameters and initial conditions. The Lorenz system is a set of three differential equations that describe the rate of change of the variables `x`, `y`, and `z`. Here's how you can implement it:

```modelica
model Lorenz
  // Parameters
  parameter Real sigma = 10;
  parameter Real rho = 28;
  parameter Real beta = 8/3;

  // Variables
  Real x(start = 1);
  Real y(start = 1);
  Real z(start = 1);

equation
  // Lorenz system equations
  der(x) = sigma * (y - x);
  der(y) = x * (rho - z) - y;
  der(z) = x * y - beta * z;
end Lorenz;
```

### Explanation:
- **Parameters**: `sigma`, `rho`, and `beta` are defined as parameters with the given values.
- **Variables**: `x`, `y`, and `z` are the state variables of the system, each initialized to 1.
- **Equations**: The `der()` function is used to define the differential equations for the Lorenz system:
  - `der(x) = sigma * (y - x)` represents the rate of change of `x`.
  - `der(y) = x * (rho - z) - y` represents the rate of change of `y`.
  - `der(z) = x * y - beta * z` represents the rate of change of `z`.

This model will simulate the chaotic behavior of the Lorenz attractor when solved using a Modelica simulation environment.
