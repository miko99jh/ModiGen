```modelica
function Rotator
  extends Modelica.Icons.Function;
  input Real x[2] "Input space phasor";
  input Modelica.Units.SI.Angle angle "Angle of rotation";
  output Real y[2] "Rotated space phasor";
protected 
  Real RotationMatrix[2, 2] = {{cos(-angle), -sin(-angle)}, 
                               {sin(-angle), cos(-angle)}};
algorithm 
  y[1] := RotationMatrix[1, 1] * x[1] + RotationMatrix[1, 2] * x[2];
  y[2] := RotationMatrix[2, 1] * x[1] + RotationMatrix[2, 2] * x[2];
end Rotator;
```

This Modelica function `Rotator` rotates a 2D space phasor by a specified angle using a rotation matrix. The function takes a 2-element real vector `x` as the space phasor and an angle in radians. It outputs the rotated space phasor `y`. The rotation matrix is defined in the protected section and is used in the algorithm section to perform the matrix multiplication.
