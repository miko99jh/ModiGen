To implement the `SpringMassDamper` model in Modelica, you can define the model with the necessary variables and equations as described. Here's how you can set it up:

```modelica
model SpringMassDamper
  // Parameters
  parameter Real k = 1;   // Spring coefficient
  parameter Real c = 0.1; // Damping coefficient
  parameter Real m = 1;   // Mass

  // Variables
  Real x(start = 1); // Displacement, initial value set to 1
  Real v;            // Velocity
  Real F;            // Resultant force

equation
  // Force balance: F = -k*x - c*v
  F = -k*x - c*v;

  // Newton's second law: m*der(v) = F
  m*der(v) = F;

  // Definition of velocity: v = der(x)
  v = der(x);
end SpringMassDamper;
```

### Explanation:
- **Parameters**: `k`, `c`, and `m` are defined as parameters with given values for the spring constant, damping coefficient, and mass, respectively.
- **Variables**: `x` is the displacement with an initial value of 1, `v` is the velocity, and `F` is the resultant force.
- **Equations**:
  - The force `F` is calculated using the spring force `-k*x` and the damping force `-c*v`.
  - Newton's second law is applied to relate the force to the acceleration (`m*der(v) = F`).
  - The velocity `v` is defined as the derivative of displacement (`v = der(x)`).

This model captures the dynamics of a simple spring-mass-damper system in a vertical direction without external forces.
