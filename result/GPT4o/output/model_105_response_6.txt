To implement a Van der Pol oscillator in Modelica with the specified parameters, you can define a model named `VanDerPol` with the parameter `lambda = 0.3`. The system will consist of two state variables, `x` and `y`, both initialized to 1. The Van der Pol oscillator is a well-known nonlinear system described by a second-order differential equation. In Modelica, you can represent this system using two first-order differential equations. Here's how you can implement it:

```modelica
model VanDerPol
  // Parameters
  parameter Real lambda = 0.3;

  // State variables
  Real x(start = 1);
  Real y(start = 1);

equation
  // Van der Pol oscillator equations
  der(x) = y;
  der(y) = lambda * (1 - x^2) * y - x;
end VanDerPol;
```

### Explanation:
- **Parameters**: The parameter `lambda` is set to 0.3, which controls the nonlinearity and damping of the system.
- **State Variables**: `x` and `y` are the state variables of the system, both initialized to 1.
- **Equations**: The equations define the derivatives of `x` and `y`:
  - `der(x) = y` represents the first derivative of `x` with respect to time.
  - `der(y) = lambda * (1 - x^2) * y - x` is the Van der Pol equation, which introduces nonlinearity through the term `(1 - x^2) * y`.

This model will simulate the behavior of a Van der Pol oscillator, capturing its characteristic nonlinear oscillations. You can simulate this model using a Modelica simulation environment to observe the dynamics of `x` and `y` over time.
