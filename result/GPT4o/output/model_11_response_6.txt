```modelica
model GreatEq
  extends IndustrialControlSystems.Logical.Comparisons.IntegerType.Interfaces.IntComparison;

  parameter Real Ts = 0 "Sample time parameter";

equation
  if Ts > 0 then
    when sample(0, Ts) then
      y = u1 >= u2;
    end when;
  else
    y = u1 >= u2;
  end if;
end GreatEq;
```

This Modelica model defines a component named `GreatEq` that performs a 'greater than or equal to' comparison between two integer inputs, `u1` and `u2`. The output `y` is a Boolean that is `true` if `u1` is greater than or equal to `u2`, and `false` otherwise. The model includes a sample time parameter `Ts`, which, when greater than 0, triggers the comparison at regular intervals; otherwise, the comparison is continuous.
