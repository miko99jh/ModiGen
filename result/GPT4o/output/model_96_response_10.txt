To create a Modelica model named `Pipe` that simulates the flow of fluid through a pipe using the Hagen-Poiseuille relationship, you can follow the structure below. This model will calculate the flow coefficient (`c`), the flow rate (`Q`), and the velocity (`v`) based on the given parameters.

```modelica
model Pipe
  // Parameters
  parameter Real P1 = 1080; // Pressure at the inlet (Pa)
  parameter Real P2 = 900;  // Pressure at the outlet (Pa)
  parameter Real L = 1;     // Length of the pipe (m)
  parameter Real D = 0.05;  // Inner diameter of the pipe (m)
  parameter Real mu = 2e-3; // Viscosity of the fluid (Pa.s)

  // Constants
  constant Real pi = 3.141592653589793;

  // Variables
  Real c;  // Flow coefficient
  Real Q;  // Flow rate (m^3/s)
  Real v;  // Velocity (m/s)
  Real deltaP; // Pressure difference (Pa)
  Real A;  // Cross-sectional area of the pipe (m^2)

equation
  // Calculate the pressure difference
  deltaP = P1 - P2;

  // Calculate the cross-sectional area of the pipe
  A = pi * (D/2)^2;

  // Calculate the flow coefficient using the Hagen-Poiseuille equation
  c = (pi * D^4) / (128 * mu * L);

  // Calculate the flow rate
  Q = c * deltaP;

  // Calculate the velocity
  v = Q / A;

end Pipe;
```

### Explanation:
- **Parameters**: These are the known values for the problem, including pressures at both ends, pipe length, diameter, and fluid viscosity.
- **Constants**: The value of π is defined for calculations involving the area and flow coefficient.
- **Variables**: These include the flow coefficient (`c`), flow rate (`Q`), velocity (`v`), pressure difference (`deltaP`), and cross-sectional area (`A`).
- **Equations**:
  - `deltaP` is the pressure difference between the two ends of the pipe.
  - `A` is the cross-sectional area of the pipe, calculated using the formula for the area of a circle.
  - `c` is the flow coefficient derived from the Hagen-Poiseuille equation.
  - `Q` is the flow rate, calculated as the product of the flow coefficient and the pressure difference.
  - `v` is the velocity of the fluid, calculated by dividing the flow rate by the cross-sectional area.

This model assumes laminar flow conditions, which is a requirement for the Hagen-Poiseuille equation to be valid.
