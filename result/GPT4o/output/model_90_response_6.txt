To implement a proportional control system in Modelica for maintaining the water level in a reservoir, you can create a simple model named `PController`. This model will simulate the change in water level over time using a proportional control approach. Below is a basic implementation:

```modelica
model PController
  // Parameters
  parameter Real target_height = 1.0; // Target water level in meters
  parameter Real Kp = 1.0; // Proportional gain

  // Variables
  Real waterLevel(start=0.2); // Initial water level in meters
  Real error; // Error between target and current water level
  Real control; // Control signal

equation
  // Calculate error
  error = target_height - waterLevel;

  // Calculate control signal using proportional control
  control = Kp * error;

  // Update water level based on control signal
  der(waterLevel) = control;

  // Simulation settings
  annotation(
    experiment(StartTime=0, StopTime=10, Tolerance=1e-6, Interval=0.1)
  );
end PController;
```

### Explanation:

1. **Parameters**:
   - `target_height`: The desired water level in the reservoir.
   - `Kp`: The proportional gain, which determines how aggressively the system responds to the error.

2. **Variables**:
   - `waterLevel`: The current water level in the reservoir, initialized to 0.2 meters.
   - `error`: The difference between the target water level and the current water level.
   - `control`: The control signal calculated using the proportional control formula.

3. **Equations**:
   - The `error` is calculated as the difference between the `target_height` and the `waterLevel`.
   - The `control` signal is calculated using the proportional gain `Kp` multiplied by the `error`.
   - The change in `waterLevel` over time (`der(waterLevel)`) is determined by the `control` signal.

4. **Simulation Settings**:
   - The simulation is set to run from time 0 to 10 seconds with a specified tolerance and interval.

This model will simulate how the water level in the reservoir changes over time as the proportional controller attempts to maintain the target water level. You can adjust the `Kp` value to see how different proportional gains affect the system's response.
