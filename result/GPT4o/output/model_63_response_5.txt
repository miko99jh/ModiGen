```modelica
model SignalVoltage
  extends Modelica.Electrical.Analog.Icons.VoltageSource;

  // Components
  Modelica.Electrical.Analog.Interfaces.PositivePin p;
  Modelica.Electrical.Analog.Interfaces.NegativePin n;
  Modelica.Blocks.Interfaces.RealInput v "Voltage between pin p and n as input signal";
  Modelica.Units.SI.Current i "Current flowing from pin p to pin n";

equation
  // Voltage between pins
  v = p.v - n.v;
  
  // Current conservation
  0 = p.i + n.i;
  
  // Current definition
  i = p.i;
end SignalVoltage;
```

This Modelica model defines a generic voltage source using an input signal for the voltage. It extends the `VoltageSource` icon from the Modelica library and includes the necessary components and equations to represent the electrical behavior of the source.
