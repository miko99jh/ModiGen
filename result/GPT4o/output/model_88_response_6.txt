To implement the `RLC1` Modelica model for the described RLC circuit, you can follow the structure below. This model will simulate the behavior of the circuit with the given parameters and variables.

```modelica
model RLC1
  // Parameters
  parameter Real R = 100;       // Resistance in ohms
  parameter Real C = 1e-3;      // Capacitance in farads
  parameter Real L = 1;         // Inductance in henrys
  parameter Real Vb = 24;       // Supply voltage in volts

  // Variables
  Real V;                       // Voltage across the parallel branch
  Real i_L;                     // Current through the inductor
  Real i_R;                     // Current through the resistor
  Real i_C;                     // Current through the capacitor

  // Equations
equation
  // Kirchhoff's voltage law for the loop
  Vb = V + L * der(i_L);

  // Current through the inductor
  i_L = i_R + i_C;

  // Ohm's law for the resistor
  i_R = V / R;

  // Capacitor current equation
  i_C = C * der(V);

  // Initial conditions
  V = 0;
  i_L = 0;
end RLC1;
```

### Explanation:
- **Parameters**: The model defines the resistance `R`, capacitance `C`, inductance `L`, and supply voltage `Vb` as parameters.
- **Variables**: The model includes variables for the voltage across the parallel branch (`V`), the current through the inductor (`i_L`), the current through the resistor (`i_R`), and the current through the capacitor (`i_C`).
- **Equations**:
  - Kirchhoff's voltage law is applied to the loop, stating that the supply voltage `Vb` is equal to the sum of the voltage across the parallel branch `V` and the voltage across the inductor `L * der(i_L)`.
  - The current through the inductor `i_L` is the sum of the currents through the resistor `i_R` and the capacitor `i_C`.
  - Ohm's law is used to relate the voltage across the resistor `V` to the current through it `i_R`.
  - The current through the capacitor `i_C` is given by the derivative of the voltage across it, multiplied by the capacitance `C`.
- **Initial Conditions**: The initial conditions for the voltage `V` and the inductor current `i_L` are set to zero.

This model can be simulated using a Modelica simulation environment to analyze the transient and steady-state behavior of the RLC circuit.
