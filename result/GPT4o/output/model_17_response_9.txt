It adds two integer inputs `u1` and `u2` and outputs their sum `y`. The model handles the addition through a conditional operation based on the sampling time `Ts`.

```modelica
model Add
  extends IndustrialControlSystems.MathOperations.IntegerType.Interfaces.IntSimpleOperation;

  parameter Real Ts = 0 "Sampling time";

  Integer u1 "First integer input";
  Integer u2 "Second integer input";

equation
  if Ts > 0 then
    when sample(0, Ts) then
      y = u1 + u2;
    end when;
  else
    y = u1 + u2;
  end if;
end Add;
```

### Explanation:
- **Extending `IntSimpleOperation`:** The model extends `IntSimpleOperation`, which means it inherits the input `u` and output `y`.
- **Parameters and Inputs:**
  - `Ts`: A parameter representing the sampling time. If `Ts` is greater than zero, the model computes the output at each sample point.
  - `u1` and `u2`: Integer inputs to be added.
- **Conditional Operation:**
  - If `Ts > 0`, the addition is performed at each sample point using the `when sample(0, Ts)` construct.
  - If `Ts <= 0`, the output `y` is directly computed as the sum of `u1` and `u2`.

This model effectively handles both continuous and discrete-time addition based on the sampling time parameter.
