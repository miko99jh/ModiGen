```modelica
model GreatEq
  extends IndustrialControlSystems.Logical.Comparisons.IntegerType.Interfaces.IntComparison;

  parameter Real Ts = 0 "Sample time, if Ts > 0, the comparison is triggered at regular intervals";

protected 
  Boolean ySampled;

equation 
  if Ts > 0 then
    when sample(0, Ts) then
      ySampled = u1 >= u2;
    end when;
    y = ySampled;
  else
    y = u1 >= u2;
  end if;

end GreatEq;
```

This Modelica model defines a component named `GreatEq` that performs a 'greater than or equal to' comparison between two integer inputs, `u1` and `u2`. The output `y` is a Boolean that is `true` if `u1` is greater than or equal to `u2`, and `false` otherwise. The model includes a sample time parameter `Ts` that, when greater than 0, triggers the comparison at regular intervals. If `Ts` is 0 or less, the comparison is continuous.
