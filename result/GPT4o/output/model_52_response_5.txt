To create a Modelica model named `ShowSaturatingInductor` that demonstrates the behavior of a SaturatingInductor component, you can use the following code. This model includes the specified parameters and components, and sets up the connections as described:

```modelica
model ShowSaturatingInductor
  // Parameters
  parameter Modelica.Units.SI.Inductance Lzer = 2 "Inductance near zero current";
  parameter Modelica.Units.SI.Inductance Lnom = 1 "Nominal inductance at nominal current";
  parameter Modelica.Units.SI.Current Inom = 1 "Nominal current";
  parameter Modelica.Units.SI.Inductance Linf = 0.5 "Inductance at large currents";
  parameter Modelica.Units.SI.Voltage U = 1.25 "Source voltage peak";
  parameter Modelica.Units.SI.Frequency f = 1/(2*Modelica.Constants.pi) "Source frequency";
  parameter Modelica.Units.SI.Angle phase = Modelica.Constants.pi/2 "Phase shift for source voltage";

  // Components
  Modelica.Electrical.Analog.Sources.SineVoltage SineVoltage1(V=U, phase=phase, f=f) "Sine voltage source";
  Modelica.Electrical.Analog.Basic.Ground Ground1 "Ground component";
  Modelica.Electrical.Analog.Basic.SaturatingInductor SaturatingInductance1(Lzer=Lzer, Lnom=Lnom, Inom=Inom, Linf=Linf, i(fixed=true)) "Saturating inductor";
  Modelica.Electrical.Analog.Basic.Inductor Inductance1(L=Lnom, i(fixed=true)) "Standard inductor";

  // Connections
  equation
    connect(SineVoltage1.n, Ground1.p);
    connect(SineVoltage1.p, SaturatingInductance1.p);
    connect(SineVoltage1.p, Inductance1.p);
    connect(SineVoltage1.n, SaturatingInductance1.n);
    connect(SineVoltage1.n, Inductance1.n);
end ShowSaturatingInductor
```

This model sets up a sine voltage source with the specified parameters and connects it to both a saturating inductor and a standard inductor in parallel. The ground component is used to complete the circuit by grounding the negative terminal of the voltage source.
