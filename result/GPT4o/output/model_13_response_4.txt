```modelica
model Xnor
  extends IndustrialControlSystems.Logical.LogicalOperations.Interfaces.LogicalBlock(
    nInput=2, final nOutput=1);

  parameter Real Ts = 0 "Sample time";

equation
  assert(nInput >= 2, "nInput must be greater than or equal to 2");

  if Ts > 0 then
    when sample(0, Ts) then
      y[1] = not Functions.Xor(u[:], nInput);
    end when;
  else
    y[1] = not Functions.Xor(u[:], nInput);
  end if;
end Xnor;
```

This Modelica model defines an `Xnor` gate that extends the `LogicalBlock` interface with two inputs and one output. It checks that the number of inputs is at least two and performs an XNOR operation on the inputs. If the sample time `Ts` is greater than zero, the operation is performed at regular intervals defined by `Ts`. Otherwise, the operation is performed continuously.
