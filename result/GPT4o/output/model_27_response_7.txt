To create a Modelica block named `EventClock` that generates a clock signal when the Boolean input changes from false to true, you can extend `Modelica.Clocked.ClockSignals.Interfaces.PartialClock` and use the `Modelica.Blocks.Interfaces.BooleanInput` for the input. Below is an example implementation:

```modelica
block EventClock
  extends Modelica.Clocked.ClockSignals.Interfaces.PartialClock;

  parameter Boolean useSolver = false "Flag to use solver method for nested clock";
  parameter String solverMethod = "default" "Solver method for nested clock";

  Modelica.Blocks.Interfaces.BooleanInput u "Boolean input signal";
  output Modelica.Clocked.ClockSignals.Clock y "Output clock signal";

protected 
  Boolean previous_u = false "Stores the previous value of u";

equation
  // Detect rising edge of the input signal
  when u and not previous_u then
    if useSolver then
      // Instantiate nested clock with solver method
      y = Modelica.Clocked.ClockSignals.Clock(solverMethod);
    else
      // Standard event clock triggered by u
      y = Modelica.Clocked.ClockSignals.Clock();
    end if;
  end when;

  // Update previous_u at each time step
  previous_u = u;
end EventClock;
```

### Explanation:
- **Parameters**:
  - `useSolver`: A Boolean parameter to decide whether to use a nested clock with a solver method.
  - `solverMethod`: A string parameter specifying the solver method for the nested clock.

- **Inputs**:
  - `u`: A Boolean input signal that triggers the clock signal.

- **Outputs**:
  - `y`: The output clock signal.

- **Protected Variables**:
  - `previous_u`: A Boolean variable to store the previous value of `u` to detect changes.

- **Equations**:
  - The `when` clause detects a rising edge in the Boolean input `u` (i.e., when `u` changes from false to true).
  - If `useSolver` is true, a nested clock is instantiated using the specified `solverMethod`.
  - Otherwise, a standard event clock is triggered directly by `u`.
  - The `previous_u` variable is updated at each time step to keep track of the last state of `u`.
