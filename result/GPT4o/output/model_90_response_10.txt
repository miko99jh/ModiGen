To implement a proportional control system in Modelica, you can create a simple model that simulates the change in water level over time. Below is a basic implementation of a Modelica model named `PController` that uses proportional control to adjust the water level in a reservoir:

```modelica
model PController
  // Parameters
  parameter Real target_height = 1.0; // Target water level in meters
  parameter Real Kp = 1.0; // Proportional gain

  // Variables
  Real waterLevel(start=0.2); // Initial water level in meters
  Real error; // Altitude error
  Real control; // Control signal

equation
  // Calculate the error
  error = target_height - waterLevel;

  // Calculate the control signal using proportional control
  control = Kp * error;

  // Update the water level based on the control signal
  der(waterLevel) = control;

  annotation(
    experiment(StartTime=0, StopTime=10, Tolerance=1e-6, Interval=0.1),
    uses(Modelica(version="3.2"))
  );
end PController;
```

### Explanation:

- **Parameters**: 
  - `target_height`: The desired water level in the reservoir.
  - `Kp`: The proportional gain, which determines how aggressively the control system responds to the error.

- **Variables**:
  - `waterLevel`: The current water level in the reservoir, initialized to 0.2 meters.
  - `error`: The difference between the target height and the current water level.
  - `control`: The control signal calculated using the proportional control formula.

- **Equations**:
  - The `error` is calculated as the difference between the `target_height` and the `waterLevel`.
  - The `control` signal is calculated by multiplying the `error` by the proportional gain `Kp`.
  - The `der(waterLevel)` equation updates the water level based on the control signal, simulating the change in water level over time.

- **Annotations**:
  - The `experiment` annotation specifies the simulation parameters, such as start time, stop time, tolerance, and interval.

This model will simulate the water level in the reservoir over time, adjusting it towards the target height using proportional control. You can run this model in a Modelica simulation environment to observe how the water level changes over time.
