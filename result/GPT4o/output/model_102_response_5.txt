To create a Modelica model named `ArrayExample` that defines an array and calculates the absolute values, minimum, maximum, and sum of its elements, you can use the following code. This code uses an equation block to perform the calculations:

```modelica
model ArrayExample
  // Define the array with initial values
  parameter Integer n = 5;
  Integer data[n] = {1, 2, 3, 4, 5};
  
  // Variables to store the results
  Integer abs1[n];
  Integer min1;
  Integer max1;
  Integer sum1;

equation
  // Calculate the absolute values of the array elements
  for i in 1:n loop
    abs1[i] = abs(data[i]);
  end for;

  // Calculate the minimum value
  min1 = min(data);

  // Calculate the maximum value
  max1 = max(data);

  // Calculate the sum of all elements
  sum1 = sum(data);

end ArrayExample;
```

### Explanation:
- **Array Definition**: The array `data` is defined with a length of `n = 5` and initialized with the values `{1, 2, 3, 4, 5}`.
- **Absolute Values**: The `abs1` array stores the absolute values of each element in `data`. Since all elements are positive, `abs1` will be the same as `data`.
- **Minimum and Maximum**: The `min1` and `max1` variables store the minimum and maximum values of the `data` array, respectively.
- **Sum**: The `sum1` variable stores the sum of all elements in the `data` array.
- **Equation Block**: The `equation` block contains a loop to calculate the absolute values and uses built-in functions `min`, `max`, and `sum` to compute the respective values.
