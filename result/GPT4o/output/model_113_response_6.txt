To create a Modelica model for an RLC circuit using a state space representation, you can follow the structure below. This model uses the `ModelicaByExample.ArrayEquations.StateSpace.LTI` component to define the state space representation of the RLC circuit. The parameters and matrices are set according to your specifications.

```modelica
model RLC
  import ModelicaByExample.ArrayEquations.StateSpace.LTI;

  // Parameters
  parameter Real Vb = 24; // Voltage source in volts
  parameter Real L = 1;   // Inductance in Henry
  parameter Real R = 100; // Resistance in ohms
  parameter Real C = 0.001; // Capacitance in Farads

  // State space matrices
  parameter Real A[2, 2] = [ -1/(R*C), 1/C; -1/L, 0 ];
  parameter Real B[2, 1] = [ 0; 1/L ];
  parameter Real Cmat[2, 2] = [ 1/R, 0; -1/R, 1 ];
  parameter Real D[2, 1] = [ 0; 0 ];

  // Initial state
  parameter Real x0[2] = {0, 0};

  // State space component
  LTI rlc_comp(
    nx=2, 
    nu=1, 
    ny=2, 
    A=A, 
    B=B, 
    C=Cmat, 
    D=D, 
    x0=x0
  );

  // Connect the input 'u' of LTI component to the voltage source
  equation
    rlc_comp.u = Vb;
end RLC;
```

### Explanation:
- **Parameters**: The model defines the parameters for the voltage source, inductor, resistor, and capacitor.
- **State Space Matrices**: The matrices `A`, `B`, `Cmat`, and `D` are defined according to the state space representation of the RLC circuit.
- **Initial State**: The initial state `x0` is set to `{0, 0}`.
- **LTI Component**: The `LTI` component is instantiated with the specified number of state variables (`nx=2`), input (`nu=1`), and outputs (`ny=2`), along with the matrices and initial state.
- **Equation**: The input `u` of the `LTI` component is connected to the voltage source `Vb`.

This model provides a state space representation of the RLC circuit, allowing for simulation and analysis within the Modelica environment.
