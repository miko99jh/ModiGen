```modelica
model NewtonCooling
  // Parameters
  parameter Temperature T_inf = 300.0 "Ambient temperature";
  parameter Temperature T0 = 280.0 "Initial temperature";
  parameter ConvectionCoefficient h = 0.7 "Convective cooling coefficient";
  parameter Area A = 1.0 "Surface area";
  parameter Mass m = 0.1 "Mass of thermal capacitance";
  parameter SpecificHeat c_p = 1.2 "Specific heat capacity";

  // Variables
  Temperature T(start = T0) "Temperature of the object";

equation
  // Newton's law of cooling
  m * c_p * der(T) = h * A * (T_inf - T);

end NewtonCooling;
```

This Modelica model defines the cooling process of an object according to Newton's law of cooling. The model includes parameters for ambient temperature, initial temperature, convective cooling coefficient, surface area, mass, and specific heat capacity. The equation governing the temperature change over time is based on the balance between the heat lost due to convection and the heat capacity of the object.
