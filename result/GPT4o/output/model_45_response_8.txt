```modelica
block ExtractSignal
  extends Modelica.ComplexBlocks.Interfaces.ComplexMIMO;

  parameter Integer extract[nout] "Indices of input signals to be extracted";

equation
  // Iterate over the range of output signals
  for i in 1:nout loop
    // Assign each specified input signal to the corresponding output
    y[i] = u[extract[i]];
  end for;
end ExtractSignal;
```

This Modelica block `ExtractSignal` is designed to extract specific signals from an input signal vector based on the indices provided in the `extract` parameter. It inherits from `Modelica.ComplexBlocks.Interfaces.ComplexMIMO`, which allows it to handle multiple input and output signals in a complex multi-input multi-output system. The `extract` parameter is an integer array that specifies which indices of the input vector `u` should be mapped to the output vector `y`. The `equation` section iterates over the range of output signals (`nout`) and assigns the specified input signals to the corresponding outputs.
