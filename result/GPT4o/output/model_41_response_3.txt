```modelica
block PI
  extends Modelica.Clocked.RealSignals.Interfaces.PartialClockedSISO;

  parameter Real kd "Gain of the discrete PI controller";
  parameter Real Td(min=Modelica.Constants.small) "Time constant of the discrete PI controller";

  output Real x(start=0) "State of the discrete PI controller";

protected 
  Real u_prev(start=0) "Previous input value for state update";

equation 
  when Clock() then
    // Integral State Update
    x = pre(x) + u / Td;
    
    // Control Output
    y = kd * (u + x);
  end when;

end PI;
```

This Modelica model defines a discrete-time PI controller. The `when Clock() then` block ensures that the state update and control output calculations occur at each clock tick, implementing the discrete-time behavior of the PI controller. The state `x` acts as an accumulator for the integral part, while the output `y` combines both the proportional and integral actions.
