```modelica
model LimitedActuator
  extends ModelicaByExample.Architectures.SensorComparison.Interfaces.Actuator;

  parameter Modelica.SIunits.Time delayTime;
  parameter Real uMax;

protected 
  Modelica.Mechanics.Rotational.Sources.Torque torque(useSupport=true);
  Modelica.Blocks.Nonlinear.Limiter limiter(uMax=uMax);
  Modelica.Blocks.Nonlinear.FixedDelay lag(delayTime=delayTime);

equation 
  // Mechanical Output
  connect(torque.flange, shaft);
  connect(torque.support, housing);

  // Signal Chain
  connect(tau, lag.u);
  connect(lag.y, limiter.u);

  // Actuation
  connect(limiter.y, torque.tau);

end LimitedActuator;
```

This Modelica model defines a `LimitedActuator` that incorporates a delay and saturation mechanism for an actuator. The model extends from a specified interface and includes parameters for delay time and maximum input signal. The components are connected to ensure that the input signal is delayed, limited, and then used to generate torque.
