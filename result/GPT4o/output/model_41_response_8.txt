```modelica
block PI
  extends Modelica.Clocked.RealSignals.Interfaces.PartialClockedSISO;

  parameter Real kd; // Gain of the discrete PI controller
  parameter Real Td(min=Modelica.Constants.small); // Time constant of the discrete PI controller

  output Real x(start=0); // State of the discrete PI controller

protected
  Real u_previous(start=0); // Previous input value

equation
  when Clock() then
    // Integral State Update
    x = x + u / Td;

    // Control Output
    y = kd * (u + x);

    // Update previous input
    u_previous = u;
  end when;
end PI;
```

This Modelica model defines a discrete-time PI controller named `PI`. It extends the `Modelica.Clocked.RealSignals.Interfaces.PartialClockedSISO` interface, which is suitable for single-input single-output (SISO) systems in a clocked (discrete-time) context. The model includes parameters for the controller gain (`kd`) and the time constant (`Td`). The internal state `x` acts as an accumulator, updated at each clock tick, and the output `y` is calculated using the PI control law.
