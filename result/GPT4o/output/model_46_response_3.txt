```modelica
block ComplexToReal
  extends Modelica.ComplexBlocks.Icons.ComplexBlock;

  // Parameters
  parameter Boolean useConjugateInput = false "If true, process the conjugate of the input";

  // Inputs
  Modelica.ComplexBlocks.Interfaces.ComplexInput u "Complex input signal";

  // Outputs
  Modelica.Blocks.Interfaces.RealOutput re "Real part of the complex input";
  Modelica.Blocks.Interfaces.RealOutput im "Imaginary part of the complex input";

equation
  // Assign real part directly
  re = Modelica.ComplexMath.real(u);

  // Assign imaginary part, negating if useConjugateInput is true
  if useConjugateInput then
    im = -Modelica.ComplexMath.imag(u);
  else
    im = Modelica.ComplexMath.imag(u);
  end if;

end ComplexToReal;
```

This Modelica block `ComplexToReal` is designed to convert a complex input signal into its Cartesian components, with the option to process the conjugate of the input. It includes a parameter to control this behavior and outputs the real and imaginary parts accordingly.
