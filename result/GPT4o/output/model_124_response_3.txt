```modelica
model AmbientCondition
  // Import the necessary Modelica library
  import Modelica.SIunits.Temperature;
  import Modelica.Thermal.HeatTransfer.Interfaces.HeatPort_a;

  // Parameter for ambient temperature
  parameter Temperature T_amb = 293.15 "Ambient temperature in Kelvin";

  // Heat transfer interface
  HeatPort_a node;

equation
  // Ensure the temperature at the node is equal to the ambient temperature
  node.T = T_amb;

  // Since this is an infinite reservoir, no heat flow occurs
  node.Q_flow = 0;
  
end AmbientCondition;
```

### Explanation:
- **Parameter `T_amb`:** This parameter represents the ambient temperature and is initialized to 293.15 K (20°C) by default. You can change this value as needed.
- **HeatPort_a `node`:** This is the heat transfer interface that allows the model to connect with other thermal components.
- **Equation `node.T = T_amb`:** This equation ensures that the temperature at the heat port is always equal to the ambient temperature.
- **Equation `node.Q_flow = 0`:** This ensures that no heat flow occurs, representing an infinite reservoir that maintains its temperature regardless of external conditions.
