To create a Modelica model named `Pipe` that simulates the flow of fluid through a pipe using the Hagen-Poiseuille relationship, you can follow the structure below. This model will calculate the flow coefficient (`c`), the flow rate (`Q`), and the velocity (`v`) based on the given parameters.

```modelica
model Pipe
  // Parameters
  parameter Real P1 = 1080; // Pressure at the inlet (Pa)
  parameter Real P2 = 900;  // Pressure at the outlet (Pa)
  parameter Real L = 1;     // Length of the pipe (m)
  parameter Real D = 0.05;  // Inner diameter of the pipe (m)
  parameter Real mu = 2e-3; // Viscosity of the fluid (Pa.s)

  // Constants
  constant Real pi = 3.141592653589793;

  // Variables
  Real c;  // Flow coefficient
  Real Q;  // Flow rate (m^3/s)
  Real v;  // Velocity (m/s)

equation
  // Calculate the flow coefficient using the Hagen-Poiseuille equation
  c = (pi * D^4) / (128 * mu * L);

  // Calculate the flow rate using the pressure difference and flow coefficient
  Q = c * (P1 - P2);

  // Calculate the velocity using the flow rate and cross-sectional area
  v = Q / (pi * (D/2)^2);

end Pipe;
```

### Explanation:

1. **Parameters**: The model defines parameters for the pressures at the inlet (`P1`) and outlet (`P2`), the length of the pipe (`L`), the inner diameter of the pipe (`D`), and the viscosity of the fluid (`mu`).

2. **Constants**: The constant `pi` is defined for calculations involving the pipe's cross-sectional area.

3. **Variables**: The model calculates three main variables:
   - `c`: The flow coefficient, which is derived from the Hagen-Poiseuille equation.
   - `Q`: The flow rate, calculated using the pressure difference and the flow coefficient.
   - `v`: The velocity of the fluid, calculated from the flow rate and the pipe's cross-sectional area.

4. **Equations**: The equations section contains the mathematical relationships used to calculate the flow coefficient, flow rate, and velocity.

This model assumes laminar flow and uses the Hagen-Poiseuille equation, which is valid for incompressible, Newtonian fluids in a circular pipe under laminar flow conditions.
