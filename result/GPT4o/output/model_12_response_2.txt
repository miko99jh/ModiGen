```modelica
model NotEqual
  extends IndustrialControlSystems.Logical.Comparisons.RealType.Interfaces.RealComparison;

  parameter Real eps = 1e-6 "Threshold for comparison";
  parameter Real Ts = 0 "Sample time, Ts > 0 for discrete sampling";

protected 
  Boolean sampleTrigger;

equation 
  if Ts > 0 then
    when sample(Ts) then
      sampleTrigger = true;
    end when;
  else
    sampleTrigger = true;
  end if;

  when sampleTrigger then
    y = abs(u1 - u2) >= eps;
  end when;

end NotEqual;
```

This Modelica model defines a `NotEqual` function that compares two real input signals, `u1` and `u2`, and outputs a Boolean `y`. The output is true if the absolute difference between `u1` and `u2` is greater than or equal to a small threshold `eps`. The comparison can be continuous or triggered at regular intervals defined by the sample time `Ts`. If `Ts` is greater than 0, the comparison is performed at discrete intervals; otherwise, it is continuous.
