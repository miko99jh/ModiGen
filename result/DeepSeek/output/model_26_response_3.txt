
Generate a Modelica block named `BooleanChange` that indicates changes in a Boolean signal within the context of clocked, non-periodic signals. The model should extend from the `Modelica.Clocked.ClockSignals.Interfaces.ClockedBlockIcon` class.
The model should include:
- A Boolean input signal 'u' of type `Modelica.Blocks.Interfaces.BooleanInput`.
- A Boolean output signal 'y' of type `Modelica.Blocks.Interfaces.BooleanOutput`.
In the equation section, determine the output 'y' by detecting changes in the input signal 'u'. On the initial activation of the clock, the output should default to 'false'. For all subsequent ticks, the output should be 'true' only if the current value of 'u' differs from its value at the most recent previous tick.

//modelica
```block BooleanChange
  extends Modelica.Clocked.ClockSignals.Interfaces.ClockedBlockIcon(
    blockName="BooleanChange");
  Modelica.Blocks.Interfaces.BooleanInput u;
  Modelica.Blocks.Interfaces.BooleanOutput y = false;
equation
  when clock then
    y = false;
  end when;
  y = if u <> pre(u) then true else false;
end BooleanChange;
```

This model will output a 'true' signal whenever the input Boolean signal changes value.
