
Generate a Modelica model named `Reactions_Array`for simulating chemical reactions using arrays. This model should accurately represent a chemical reaction dynamics by using arrays to handle multiple reaction components simultaneously. The model inputs include concentrations of chemicals as a real array 'C[3]', and reaction rate constants as a parameter array 'k[3]' with default values {0.1, 0.1, 10}. The model should initiate with an initial concentration array 'C = {1, 1, 0}'. The output should be the rate of change of concentrations, governed by the differential equations set in the 'equation' section. The value of 'der(C)' is determined by '{-k[1]*C[1]*C[2] + k[2]*C[3], -k[1]*C[1]*C[2] + k[2]*C[3] - k[3]*C[2]*C[3], k[1]*C[1]*C[2] -k [2]*C[3] -k [3]*C[2]*C[3]}'.

//modelica
```model Reactions_Array
  parameter Real C[3](start={1, 1, 0}) "Concentration of chemicals";
  parameter Real k[3](default={0.1, 0.1, 10}) "Reaction rate constants";
  Real derC[3] "Rate of change of concentration";
equation
  derC = {-k[1]*C[1]*C[2] + k[2]*C[3], -k[1]*C[1]*C[2] + k[2]*C[3] - k[3]*C[2]*C[3], k[1]*C[1]*C[2] - k[2]*C[3] - k[3]*C[2]*C[3]};
end Reactions_Array;
```
