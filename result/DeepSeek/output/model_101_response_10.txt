
Implement a Modelica function named `SphereProperties` to calculate the surface area (`surface`) and volume (`volume`) of a sphere based on its radius (`radius`). The function should use an algorithm block to implement the calculation logic and utilize the pi constant from the standard Modelica library (Modelica.Constants.pi) for the formulas.

//modelica
```function SphereProperties
  input Real radius;
  output Real surface;
  output Real volume;
algorithm
  surface := 4 * Modelica.Constants.pi * radius^2;
  volume := 4/3 * Modelica.Constants.pi * radius^3;
end SphereProperties;```

This function takes the radius of the sphere as input and calculates the surface area and volume of the sphere. The surface area is calculated using the formula 4 * pi * radius^2, and the volume is calculated using the formula 4/3 * pi * radius^3. The results are returned as output parameters.

Note that the function name should be in camel case (i.e., SphereProperties), and the input and output variables should be defined with appropriate types (i.e., Real for radius, surface, and volume). The algorithm block should contain the calculation logic for surface and volume.

In the algorithm block, the surface area and volume are calculated using the given formulas and the pi constant from the Modelica.Constants package. The results are then assigned to the output variables surface and volume.
