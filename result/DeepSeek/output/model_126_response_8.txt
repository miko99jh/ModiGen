
Generate a Modelica model named `FirstOrderExperiment` that defines first order equation with experiment conditions for a simple system. The model should include a real variable 'x' and initialize 'x=2'. This variable should dynamically change based on the differential equation defined, aiming for the value of 'x' to converge towards 1.0.

//modelica
```model FirstOrderExperiment
  parameter Real x(start=2) "State variable";
  parameter Real k(start=0.1) "Rate constant";
equation
  der(x) = -k*x;
  when x < 1.0 then
    terminate("Experiment complete");
  end when;
end FirstOrderExperiment;
```

This model defines a first order differential equation where the rate of change of 'x' is proportional to its own negative value. The parameter 'k' is the rate constant that determines the rate of decay of 'x'. The experiment is terminated when 'x' is less than 1.0.

The model is initialized with 'x' set to 2.0 and 'k' set to 0.1. The goal of the experiment is to observe how 'x' decreases over time until it reaches a value less than 1.0.

Please note that this model does not include any time-varying inputs or outputs, and it is assumed that the experiment duration is long enough for 'x' to reach its steady state value.

The model is simple and does not include any complex components or algorithms. It is designed to demonstrate the basic principles of modeling and simulation in Modelica.

To simulate this model, you can use any Modelica simulation software that supports the Modelica language.
