
To solve the sum (`sum`) of all integers from 1 to n (where n is 10), please help me implement a Modelica model named `Summation`. The model should use an algorithm block with a for loop to perform the accumulation, ensuring that sum is initialized before the loop.

//modelica
```model Summation
  parameter Integer n = 10;
  Real sum;
algorithm 
  sum := 0;
  for i in 1:n loop
    sum := sum + i;
  end for;
end Summation;
```

This Modelica model named `Summation` calculates the sum of all integers from 1 to n using a for loop in the algorithm section. The sum is initialized to 0 before the loop starts, and in each iteration of the loop, the current value of sum is updated by adding the current value of i. The loop iterates from 1 to n, inclusive.

Please note that the Modelica language does not support a direct syntax for summing a range of integers. However, you can achieve this by using a for loop to iterate over the range and update the sum in each iteration.
