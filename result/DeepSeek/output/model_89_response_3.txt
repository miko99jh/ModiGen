
Please use modelica to help me implement a model named `NewtonCoolingDynamic`: Simulate the temperature change of a water under Newton's law of cooling. Among them, the initial temperature of water (`T0`) is 100 °C, the ambient temperature (`T_inf`) is 25 °C, the water meter area (`A`) is 1 square meter, the mass (`m`) is 1kg, the specific heat (`c_p`) is 4200 J/(kg.°C), and the convection cooling coefficient (`h`) is 767 W/(m2.°C). Use an initial equation to set `T` to `T0`. 20 seconds after the start of the simulation, the ambient temperature suddenly dropped to 0°C.

//modelica
```model NewtonCoolingDynamic
  parameter Real T0(unit = "degC") = 100; // Initial temperature of water
  parameter Real T_inf(unit = "degC") = 25; // Ambient temperature
  parameter Real A(unit = "m^2") = 1; // Water meter area
  parameter Real m(unit = "kg") = 1; // Mass
  parameter Real c_p(unit = "J/(kg.degC)") = 4200; // Specific heat
  parameter Real h(unit = "W/(m^2.degC)") = 767; // Convection cooling coefficient
  Real T(start = T0, unit = "degC"); // Temperature of water
equation
  der(T) = (T_inf - T) / (A * h); // Newton's law of cooling
  when time = 20 "s" then
    T_inf = 0 "degC"; // Ambient temperature suddenly dropped to 0°C
  end when;
end NewtonCoolingDynamic;
```
