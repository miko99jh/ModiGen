
Generate a Modelica block named `FromPolar` that converts a space phasor from polar coordinates to Cartesian coordinates. The block should include the following components and details:
1. Components:
-  The block should extend from Modelica.Blocks.Interfaces.MIMOs with final n set to 2.
2. Equations:
Implement the coordinate transformation where the input vector 'u' represents polar components and the output vector 'y' represents Cartesian components:
The first input 'u[1]' shall be treated as the magnitude of the space phasor.
The second input 'u[2]' shall be treated as the angle of the space phasor.
The output vector 'y' should be defined as the projection of the magnitude onto the Cartesian axes using the cosine and sine of the input angle.

Please ensure the block is syntactically correct and adheres to the Modelica language standard.

//modelica
```block FromPolar extends Modelica.Blocks.Interfaces.MIMOs(n=2);
  input Real u[2] "Polar components (magnitude, angle)";
  output Real y[2] "Cartesian components";
equation
  y[1] = u[1] * cos(u[2]);
  y[2] = u[1] * sin(u[2]);
end FromPolar;
```

This block converts a space phasor from polar coordinates (magnitude, angle) to Cartesian coordinates (x, y). The first output 'y[1]' is the projection of the magnitude onto the x-axis, and the second output 'y[2]' is the projection of the magnitude onto the y-axis.
