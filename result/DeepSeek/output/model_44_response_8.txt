
Generate a Modelica model that outputs the hyperbolic tangent of a complex input signal. The model should extend the `Modelica.ComplexBlocks.Interfaces.ComplexSISO` interface. The input is a complex number 'uInternal', and the output 'y' should be the hyperbolic tangent of this input. The model should be named `Tanh`. Use the `Modelica.ComplexMath.tanh` function to calculate the output.

//modelica
```block Tanh
model Tanh
  extends Modelica.ComplexBlocks.Interfaces.ComplexSISO;
  input Complex uInternal(unit="1") "Complex input signal";
  output Complex y(unit="1") "Complex output signal";
algorithm 
  y := Modelica.ComplexMath.tanh(uInternal);
end Tanh;
```

This model takes a complex input signal, 'uInternal', and applies the hyperbolic tangent function to it, resulting in the complex output signal, 'y'. The Modelica.ComplexMath.tanh function is used to calculate the hyperbolic tangent of the input.

Note: The `unit="1"` annotation is used to specify the unit of the input and output signals. In this case, the unit is "1", which means the signals are dimensionless.
