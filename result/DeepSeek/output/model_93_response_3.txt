
We have a container, and we want to keep the height of the water (`waterLevel`) in the container at a target height (`target_height`) of 1 meter. The water height (`waterLevel`) in the pool starts at 0.2m, and the pool will leak 0.1m per unit time. Use PID control to implement the above control system (defining the height error as `error`, and splitting the logic into proportional control `Pcontrol` which handles the leak, integral control `Icontrol`, and differential control `Dcontrol`) and simulate the change of the water level in the pool over time with a proportional coefficient (`Kp`) of 1, an integral coefficient (`Ki`) of 0.2, and a differential coefficient (`Kd`) of 0.1. Please help me implement a modelica model named `PIDController`.

//modelica
```model PIDController
  parameter Real Kp = 1 "Proportional coefficient";
  parameter Real Ki = 0.2 "Integral coefficient";
  parameter Real Kd = 0.1 "Differential coefficient";
  parameter Real target_height = 1 "Target height";
  parameter Real initial_waterLevel = 0.2 "Initial water level";
  parameter Real leak_rate = 0.1 "Leak rate";
  Real waterLevel(start = initial_waterLevel) "Water level";
  Real error "Error";
  Real Pcontrol "Proportional control";
  Real Icontrol "Integral control";
  Real Dcontrol "Differential control";
  Real integral(start = 0) "Integral term";
  Real derivative(start = 0) "Derivative term";
  Real previous_error(start = 0) "Previous error";
equation
  // Calculate the error
  error = target_height - waterLevel;
  
  // Calculate the proportional control
  Pcontrol = Kp * error;
  
  // Calculate the integral control
  integral = integral + Ki * error * time;
  Icontrol = integral;
  
  // Calculate the derivative control
  derivative = Kd * (error - previous_error) / time;
  Dcontrol = derivative;
  
  // Update the water level
  der(waterLevel) = Pcontrol + Icontrol + Dcontrol - leak_rate;
  
  // Update the previous error
  previous_error = error;
end PIDController;
```

This model will simulate the change of the water level in the pool over time. The proportional control `Pcontrol` handles the leak, the integral control `Icontrol` accumulates the error over time, and the differential control `Dcontrol` anticipates future changes in the error. The water level is updated based on the sum of these controls and the leak rate. The previous error is used to calculate the derivative control.

Please note that this model assumes that the leak rate is constant and does not change over time. If the leak rate changes over time, you would need to modify the model to account for this.

Also, this model does not include any constraints or limits on the water level. If the water level goes below 0 or above the target height, you would need to add logic to handle these cases.
