1. `Modelica.Blocks.Interfaces.BooleanInput fire` to define the Boolean input for firing the thyristor.

In the equation section, define the following behavior:
- The thyristor is off (`off=true`) when the semiconductor state variable `s` is less than 0 or when the `fire` signal is false (`not fire`).

Provide the necessary equations to represent the functionality of the ideal GTO thyristor, where the thyristor's state is controlled by the semiconductor's condition and the `fire` signal.

//modelica
```model IdealGTOThyristor
  extends Modelica.Electrical.Analog.Interfaces.IdealSemiconductor;
  Modelica.Blocks.Interfaces.BooleanInput fire;

equation
  off = (s < 0) or not fire;
end IdealGTOThyristor;
```

This model describes an ideal GTO thyristor that is turned off when the semiconductor state variable `s` is less than 0 or when the `fire` signal is false. The `fire` signal is used to control the conduction behavior of the thyristor. When the `fire` signal is true, the thyristor can conduct current in the forward direction.
