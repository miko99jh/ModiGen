Set the initial angle of `theta` to 1.5 rad and the initial angular velocity of `omega` to 0 rad/s. Use a gravity constant g=9.81 m/s².

//modelica
```model SimplePendulum
  parameter Real L=1 "Length of the pendulum";
  parameter Real m=1 "Mass of the pendulum";
  parameter Real g=9.81 "Gravity constant";
  Real theta(start=1.5) "Angle of the pendulum";
  Real omega(start=0) "Angular velocity of the pendulum";
  Real F "Force applied to the pendulum";
equation
  // Newton's second law for rotation
  m*L^2*der(omega) = -m*g*L*sin(theta) + F;
  // Angular displacement equation
  der(theta) = omega;
end SimplePendulum;
```

The model uses the following variables:

- `theta`: The angle of the pendulum with respect to the vertical.
- `omega`: The angular velocity of the pendulum.
- `F`: The force applied to the pendulum.

The model uses the following differential equations:

- `der(omega) = -g*sin(theta)/L + F/m/L^2`: The equation of motion for the pendulum.
- `der(theta) = omega`: The equation for the angular displacement of the pendulum.
