
Generate a Modelica block named `Impedance` that calculates the complex impedance of a battery. The model should include the following components and details:
1.Model Functionality: The model calculates the complex impedance based on the given frequency and cell data.
2.Components:
- The model should extend `Modelica.Blocks.Icons.Block`.
- Import the necessary constants and functions from Modelica.Constants and Modelica.ComplexMath.
- Include a parameter for transient cell data: `Modelica.Electrical.Batteries.ParameterRecords.TransientData.CellData cellData`.
- Include a RealInput for frequency with unit Hz: Modelica.Blocks.Interfaces.RealInput f.
- Include a ComplexOutput for the complex impedance with units Ohm for both real and imaginary parts: `Modelica.ComplexBlocks.Interfaces.ComplexOutput z(re, im)`.
3.Equations and Expressions:
 The model should include the following equation to calculate the complex impedance:
 `z = Complex(cellData.R0, 0) + sum({1/Complex(1/cellData.rcData[k].R, 2*pi*f*cellData.rcData[k].C) for k in 1:cellData.nRC})`.

Please ensure that the model is syntactically correct and adheres to the Modelica language standard.

//modelica
```block Impedance extends Modelica.Blocks.Icons.Block
  import Modelica.Constants;
  import Modelica.ComplexMath;
  import Modelica.Electrical.Batteries.ParameterRecords.TransientData.CellData;
  import Modelica.Blocks.Interfaces;
  import Modelica.ComplexBlocks.Interfaces;

  parameter CellData cellData;
  RealInput f(unit="Hz") "Frequency";
  ComplexOutput z(re(unit="Ohm"), im(unit="Ohm")) "Complex impedance";

equation
  z = Complex(cellData.R0, 0) + sum({1/Complex(1/cellData.rcData[k].R, 2*pi*f*cellData.rcData[k].C) for k in 1:cellData.nRC});
end Impedance;
```
This model calculates the complex impedance of a battery based on the given frequency and cell data. The impedance is calculated as the sum of the resistance and the reciprocal of the sum of the complex resistances of the RC elements.
