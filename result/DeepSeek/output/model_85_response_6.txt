
Create a Modelica model named `SignalCurrent` to serve as a polyphase signal current source. The model should include the following components and details:
1. Components:
- Parameter for the number of phases: parameter Integer m(min=1) = 3.
- Voltage drops between the two plugs: Modelica.Units.SI.Voltage v[m] = plug_p.pin.v - plug_n.pin.v .
- Positive polyphase plug: Modelica.Electrical.Polyphase.Interfaces.PositivePlug plug_p(final m=m).
- Negative polyphase plug: Modelica.Electrical.Polyphase.Interfaces.NegativePlug plug_n(final m=m).
- Real input for the current: Modelica.Blocks.Interfaces.RealInput i[m] with each unit="A".
- Array of signal current sources: Modelica.Electrical.Analog.Sources.SignalCurrent signalCurrent[m].
2. Equations:
 The model should include the following connections:
-  Signal Routing: Route each element of the external input signal vector 'i' to the corresponding control input  ('i') of the 'signalCurrent' array.
-  Electrical Connection: Interpose the current source array between the plugs such that the positive terminals ('p') of the sources connect to the internal pins of the positive plug ('plug_p.pin'), and the negative terminals ('n') connect to the internal pins of the negative plug ('plug_n.pin').

Please ensure that the model is syntactically correct and adheres to the Modelica language standard.

//modelica
```model SignalCurrent
  parameter Integer m(min=1) = 3;
  Modelica.Units.SI.Voltage v[m] = plug_p.pin.v - plug_n.pin.v;
  Modelica.Electrical.Polyphase.Interfaces.PositivePlug plug_p(final m=m);
  Modelica.Electrical.Polyphase.Interfaces.NegativePlug plug_n(final m=m);
  Modelica.Blocks.Interfaces.RealInput i[m] "Current" each unit="A";
  Modelica.Electrical.Analog.Sources.SignalCurrent signalCurrent[m](i=i);
equation 
  for i in 1:m loop
    plug_p.pin.i[i] = signalCurrent[i].i;
    plug_n.pin.i[i] = -signalCurrent[i].i;
  end for;
end SignalCurrent;
```

This model creates a polyphase signal current source that can be used in a simulation environment. The number of phases is defined by the parameter'm'. The voltage drop between the two plugs is calculated as the difference between the voltages at the positive and negative plug. The external input signal vector 'i' is routed to the corresponding control input of the'signalCurrent' array. The current sources are interposed between the plugs, with the positive terminals of the sources connecting to the internal pins of the positive plug and the negative terminals connecting to the internal pins of the negative plug.

Please note that this model does not include any checks for the validity of the input signals or the connection of the plugs. It is assumed that these checks are performed elsewhere in the simulation environment.
