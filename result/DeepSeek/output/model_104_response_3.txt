
To solve the sum (`sum`) of all integers from 1 to n (where n is 10), please help me implement a Modelica model named `Summation`. The model should use an algorithm block with a for loop to perform the accumulation, ensuring that sum is initialized before the loop.

//modelica
```model Summation
  parameter Integer n = 10;
  Real sum;
algorithm 
  sum := 0;
  for i in 1:n loop
    sum := sum + i;
  end for;
end Summation;
```

This Modelica model named `Summation` calculates the sum of all integers from 1 to n using a for loop. The parameter `n` is set to 10 by default, but it can be changed as needed. The algorithm block initializes `sum` to 0, then iterates from 1 to `n`, adding each integer to `sum`. The result is stored in the variable `sum`, which can be accessed after the simulation.
