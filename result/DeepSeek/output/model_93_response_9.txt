
We have a container, and we want to keep the height of the water (`waterLevel`) in the container at a target height (`target_height`) of 1 meter. The water height (`waterLevel`) in the pool starts at 0.2m, and the pool will leak 0.1m per unit time. Use PID control to implement the above control system (defining the height error as `error`, and splitting the logic into proportional control `Pcontrol` which handles the leak, integral control `Icontrol`, and differential control `Dcontrol`) and simulate the change of the water level in the pool over time with a proportional coefficient (`Kp`) of 1, an integral coefficient (`Ki`) of 0.2, and a differential coefficient (`Kd`) of 0.1. Please help me implement a modelica model named `PIDController`.

//modelica
```model PIDController
  parameter Real Kp = 1 "Proportional coefficient";
  parameter Real Ki = 0.2 "Integral coefficient";
  parameter Real Kd = 0.1 "Differential coefficient";
  parameter Real target_height = 1 "Target height";
  parameter Real waterLevel_start = 0.2 "Initial water level";
  parameter Real leak_rate = 0.1 "Leak rate";
  Real error(start = target_height - waterLevel_start) "Height error";
  Real integral(start = 0) "Integral term";
  Real derivative(start = 0) "Derivative term";
  Real control(start = 0) "Control output";
equation
  // Proportional control
  Pcontrol = Kp * error;
  // Integral control
  integral = integral + Ki * error * timeStep;
  Icontrol = integral;
  // Differential control
  derivative = Kd * (error - derivative) / timeStep;
  Dcontrol = derivative;
  // Total control
  control = Pcontrol + Icontrol + Dcontrol;
  // Update water level
  der(waterLevel) = control - leak_rate;
  // Update error
  error = target_height - waterLevel;
end PIDController;
```

This model uses the PID control algorithm to adjust the water level in the pool. The proportional control (`Pcontrol`) handles the leak by decreasing the water level. The integral control (`Icontrol`) accumulates the error over time and increases the water level to reduce the error. The differential control (`Dcontrol`) anticipates future errors based on the rate of change of the error and adjusts the water level accordingly. The total control output is the sum of these three controls. The water level in the pool is updated by subtracting the leak rate and adding the control output. The error is updated at each time step to reflect the current difference between the target height and the current water level.
