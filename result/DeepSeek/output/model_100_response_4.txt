
Implement a Modelica function named `CircleProperties` to calculate the circumference (`circumference`) and area (`area`) of a circle based on its radius (`radius`). The function should include a protected section to define the diameter (`diameter`) and use an algorithm block to implement the logic for both outputs. Assume the value of pi is 3.14159 for the calculations.

//modelica
```function CircleProperties
  input Real radius;
  output Real circumference;
  output Real area;
protected 
  Real diameter;
algorithm 
  diameter := 2 * radius;
  circumference := 2 * 3.14159 * radius;
  area := 3.14159 * radius * radius;
end CircleProperties;```

This Modelica function takes the radius of a circle as input and returns the circumference and area of the circle as output. The protected section defines the diameter of the circle, which is calculated as twice the radius. The algorithm section then calculates the circumference and area of the circle using the formula for these properties.

The function is used in the following way:

```modelica
model TestCircleProperties
  Real radius = 5;
  Real circumference;
  Real area;
equation 
  (circumference, area) = CircleProperties(radius);
end TestCircleProperties;
```
