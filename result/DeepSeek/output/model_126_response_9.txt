
Generate a Modelica model named `FirstOrderExperiment` that defines first order equation with experiment conditions for a simple system. The model should include a real variable 'x' and initialize 'x=2'. This variable should dynamically change based on the differential equation defined, aiming for the value of 'x' to converge towards 1.0.

//modelica
```model FirstOrderExperiment
  parameter Real k = 0.5 "Rate constant";
  Real x(start=2) "Variable to be converged";
equation
  der(x) = -k*x;
  when x < 1.0 then
    terminate("Experiment complete");
  end when;
end FirstOrderExperiment;```

This Modelica model defines a first-order differential equation where the variable 'x' is decreasing over time due to the negative feedback effect of the rate constant 'k'. The experiment is terminated when 'x' is less than 1.0. The initial value of 'x' is set to 2.0.

The Modelica model is saved as a.mo file named `FirstOrderExperiment.mo`.

The Modelica model can be used to simulate the system under different conditions and analyze the convergence of 'x' towards 1.0.
