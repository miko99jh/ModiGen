
Generate a Modelica model named `IdealCore` that represents an ideal transformer with 3 windings. The model should include the following components and details:
 1.Components:
- The model should extend `PartialCore`.
2. Equations and Expressions:
 The model should include the following equations to define the relationships between the voltages and currents of the windings:
- The magnetizing current vector 'im' should be set to zero for all 'm' phases using the 'zeros()' function.
- Define the voltage transformation ratios: 'v1 = n12 * v2' and 'v1 = n13 * v3'.

Please ensure that the model is syntactically correct and adheres to the Modelica language standard.

//modelica
```model IdealCore
  extends Modelica.Electrical.Machines.Core.PartialCore;
  parameter Integer n12(start=1) "Voltage transformation ratio 1 to 2";
  parameter Integer n13(start=1) "Voltage transformation ratio 1 to 3";
  parameter Integer m(start=1) "Number of phases";
  parameter Real v1(start=1) "Primary voltage";
  parameter Real v2(start=1) "Secondary voltage";
  parameter Real v3(start=1) "Tertiary voltage";
  Real im[m](start=zeros(m)) "Magnetizing current";
equation 
  im = zeros(m);
  v1 = n12 * v2;
  v1 = n13 * v3;
end IdealCore;
```

This model is an ideal transformer with 3 windings. The magnetizing current is set to zero for all phases. The voltage transformation ratios are defined by the parameters 'n12' and 'n13'. The primary voltage 'v1' is equal to the product of the voltage transformation ratios and the secondary and tertiary voltages 'v2' and 'v3' respectively.
