
Implement the Modelica function named `Rt_T` for solving the temperature characteristics of platinum resistance. The input is the current temperature (`T`) and the output is the resistance value (`Rt`) of the platinum resistor. It is known that the reference resistance at 0°C is `R0` = 100 ohms. The function should use an `algorithm` block and define `R0` along with constants `A` = 3.9083*10^(-3), `B` = -5.775*10^(-7), and `C` = -4.183*10^(-12) as protected internal variables. When (`T` >= 0) and (`T` < 850), the resistance is calculated as `Rt` = R0 * (1 + A*T + B*T^2); When (`T` > -200) and (`T` < 0), the resistance is calculated as `Rt` = R0 * (1 + A*T + B*T^2 + C*T^3 * (T - 100)).

//modelica
```function Rt_T(Real T) "Temperature characteristics of platinum resistance" input Real T "Current temperature"; output Real Rt "Resistance value"; protected Real R0 = 100 "Reference resistance at 0°C"; Real A = 3.9083*10^(-3) "Coefficient A"; Real B = -5.775*10^(-7) "Coefficient B"; Real C = -4.183*10^(-12) "Coefficient C"; algorithm if (T >= 0) and (T < 850) then Rt := R0 * (1 + A*T + B*T^2); elseif (T > -200) and (T < 0) then Rt := R0 * (1 + A*T + B*T^2 + C*T^3 * (T - 100)); else Rt := R0; end if; end Rt_T;```

This function will be used in the model of a platinum resistance thermometer (PRT) to calculate the resistance of the resistor at a given temperature.

The function is defined as follows:

- `Rt_T` is the name of the function.
- `Real T` is the input parameter representing the current temperature.
- `Real Rt` is the output parameter representing the resistance value of the platinum resistor.
- `R0`, `A`, `B`, and `C` are the protected internal variables representing the reference resistance at 0°C, the coefficients of the temperature dependence of the resistance, respectively.
- The `algorithm` block contains the logic of the function. It calculates the resistance value based on the temperature range.
- The `if` statement checks the temperature range and calculates the resistance value accordingly.
- The `:=` operator is used to assign the calculated resistance value to the output parameter `Rt`.
- The `else` clause is used to handle the case when the temperature is outside the defined ranges. In this case, the resistance is set to the reference resistance `R0`.

This function can be used in a model of a platinum resistance thermometer to calculate the resistance of the resistor at a given temperature.
