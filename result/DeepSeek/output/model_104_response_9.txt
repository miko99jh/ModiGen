
To solve the sum (`sum`) of all integers from 1 to n (where n is 10), please help me implement a Modelica model named `Summation`. The model should use an algorithm block with a for loop to perform the accumulation, ensuring that sum is initialized before the loop.

//modelica
```model Summation
  parameter Integer n = 10;
  Real sum;
algorithm 
  sum := 0;
  for i in 1:n loop
    sum := sum + i;
  end for;
end Summation;
```

This Modelica model named `Summation` calculates the sum of all integers from 1 to n using a for loop and an algorithm block. The parameter `n` is set to 10, but it can be changed to any other integer value. The variable `sum` is initialized to 0 before the loop starts, and in each iteration of the loop, `sum` is incremented by the current value of `i`. After the loop finishes, the final value of `sum` is the sum of all integers from 1 to n.

Please note that this model assumes that the parameter `n` is a positive integer. If `n` is not a positive integer, the model may not behave as expected.
