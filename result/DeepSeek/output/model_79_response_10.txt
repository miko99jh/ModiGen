
Generate a Modelica block named `VfController` that implements a Voltage-Frequency-Controller. The block should include the following components and details:
1. Block Functionality: The block controls the voltage and frequency for a multi-phase system.
2. Components:
- The block should extend from Modelica.Blocks.Interfaces.SIMO with u having the unit "Hz" and final nout set to m.
- Parameter for the number of phases: parameter Integer m=3. 
- Parameter for the orientation of phases: parameter Modelica.Units.SI.Angle orientation[m] initialized with -Modelica.Electrical.Polyphase.Functions.symmetricOrientation(m).
- Parameter for the nominal RMS voltage per phase: parameter Modelica.Units.SI.Voltage VNominal.
- Parameter for the nominal frequency: parameter Modelica.Units.SI.Frequency fNominal.
- Parameter for the common phase shift: parameter Modelica.Units.SI.Angle BasePhase=0.
- Parameter for economy mode: parameter Boolean EconomyMode=false.
- Output for the integrator state: output Modelica.Units.SI.Angle x(start=0, fixed=true).
- Output for the voltage amplitude: output Modelica.Units.SI.Voltage amplitude.
3. Protected Variables:
- Integer pow initialized as if EconomyMode then 2 else 1.
4. Equations:
 - Voltage Modulation: Calculate the 'amplitude' as the peak value (based on 'VNominal'). For input frequencies below the nominal value, the amplitude should scale based on the ratio of actual to nominal frequency raised to the power of 'pow'. Above nominal frequency, it remains at peak nominal voltage.
- Phase Integration: The derivative of the internal state 'x' must be proportional to the input frequency converted to angular velocity.
- Multi-phase Synthesis: Generate the output vector 'y' as a set of sine waves, using the calculated 'amplitude' and a phase argument that combines the state 'x', the 'BasePhase', and the specific phase 'orientation'.

Please ensure that the block is syntactically correct and adheres to the Modelica language standard.

//modelica
```block VfController extends Modelica.Blocks.Interfaces.SIMO(final nout=m, unit="Hz");
  parameter Integer m=3;
  parameter Modelica.Units.SI.Angle orientation[m] = -Modelica.Electrical.Polyphase.Functions.symmetricOrientation(m);
  parameter Modelica.Units.SI.Voltage VNominal;
  parameter Modelica.Units.SI.Frequency fNominal;
  parameter Modelica.Units.SI.Angle BasePhase=0;
  parameter Boolean EconomyMode=false;
  output Modelica.Units.SI.Angle x(start=0, fixed=true);
  output Modelica.Units.SI.Voltage amplitude;
  protected 
  Integer pow if EconomyMode then 2 else 1;
  equation 
  amplitude = if u < fNominal then VNominal * (u/fNominal)^pow else VNominal;
  der(x) = Modelica.Units.SI.Frequency.convert(u, "rad/s");
  y = Modelica.Electrical.Polyphase.Variables.voltage(amplitude, x + BasePhase, orientation);
end VfController;```
