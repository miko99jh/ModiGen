
Generate a Modelica model of a Mixing Unit named `MixingUnit`. The functionality of the model is to simulate the mixing of substances with cooling temperature input and provide outputs for concentration and temperature. The inputs and outputs are: 
- `RealInput T_c` (unit: K), representing the cooling temperature. 
- `RealOutput c` (unit: mol/l), representing the concentration. 
- `RealOutput T` (unit: K), representing the temperature inside the mixing unit. 

The model includes several parameters: 
- `Real c0 = 0.848` (unit: mol/l), the nominal concentration. 
- `Modelica.Units.SI.Temperature T0 = 308.5`, the nominal temperature. 
- `Real a1 = 0.2674`, `a21 = 1.815`, `a22 = 0.4682`, `b = 1.5476`, `k0 = 1.05e14`, `eps = 34.2894`, all of which are process parameters.

The protected variables include: 
- `Modelica.Units.SI.Time tau0 = 60`, a protected parameter related to time constants in the system.
- `Real wk0 = k0/c0`, `weps = eps*T0`, `wa11 = a1/tau0`, `wa12 = c0/tau0`, `wa13 = c0*a1/tau0`, `wa21 = a21/tau0`, `wa22 = a22*T0/tau0`, `wa23 = T0*(a21 - b)/tau0`, `wb = b/tau0`.

Equations & Physics: 
- Reaction Kinetics: Define the reaction speed 'gamma' based on the concentration 'c' and an Arrhenius-like temperature dependence using the pre-calculated frequency factor 'wk0' and activation energy constant 'weps'.
- Mass Balance: Implement the differential equation for concentration 'c', considering a dilution/feed term ('wa13'), a linear decay term ('wa11*c'), and the consumption due to reaction ('wa12*gamma').
- Energy Balance: Implement the differential equation for temperature 'T', accounting for heat of reaction ('wa22*gamma'), heat loss/gain via flow ('wa21*T'), and the influence of the external cooling temperature 'T_c' scaled by the coefficient 'wb'.

//modelica
```model MixingUnit
  parameter Modelica.Units.SI.Temperature T0 = 308.5 "Nominal temperature";
  parameter Real c0 = 0.848 "Nominal concentration";
  parameter Real a1 = 0.2674;
  parameter Real a21 = 1.815;
  parameter Real a22 = 0.4682;
  parameter Real b = 1.5476;
  parameter Real k0 = 1.05e14;
  parameter Real eps = 34.2894;
  parameter Modelica.Units.SI.Time tau0 = 60 "Time constant";
  Real wk0;
  Real weps;
  Real wa11;
  Real wa12;
  Real wa13;
  Real wa21;
  Real wa22;
  Real wa23;
  Real wb;
  RealInput T_c "Cooling temperature";
  RealOutput c "Concentration";
  RealOutput T "Temperature";
initial equation 
  wk0 = k0/c0;
  weps = eps*T0;
  wa11 = a1/tau0;
  wa12 = c0/tau0;
  wa13 = c0*a1/tau0;
  wa21 = a21/tau0;
  wa22 = a22*T0/tau0;
  wa23 = T0*(a21 - b)/tau0;
  wb = b/tau0;
equation 
  // Reaction Kinetics
  Real gamma = wk0*exp(-weps/(R*T))*c*T_c;
  // Mass Balance
  der(c) = wa13 - wa11*c - wa12*gamma;
  // Energy Balance
  der(T) = wa22*gamma - wa21*T + wb*T_c;
end MixingUnit;
```

This model will be used to simulate the mixing process in a chemical reactor.
