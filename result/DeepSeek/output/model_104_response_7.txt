
To solve the sum (`sum`) of all integers from 1 to n (where n is 10), please help me implement a Modelica model named `Summation`. The model should use an algorithm block with a for loop to perform the accumulation, ensuring that sum is initialized before the loop.

//modelica
```model Summation
  parameter Integer n = 10;
  Real sum = 0;
algorithm 
  for i in 1:n loop
    sum := sum + i;
  end for;
end Summation;
```

This Modelica model named `Summation` calculates the sum of all integers from 1 to n using a for loop in the algorithm section. The sum is initialized to 0 before the loop starts. In each iteration of the loop, the current value of sum is updated by adding the current value of i. The loop continues until all integers from 1 to n have been added to sum.

Please note that this model assumes that n is a positive integer. If n is not a positive integer, the model may not behave as expected.

To use this model, you would instantiate it in a Modelica model and run the simulation. The final value of sum would be available after the simulation has completed.

For example, if you instantiate the `Summation` model with n = 10, the sum would be 55.

This model is a simple example of a Modelica model that uses a for loop to perform a calculation. It demonstrates how to initialize a variable, use a for loop to iterate over a range of values, and update the value of a variable in each iteration of the loop.
