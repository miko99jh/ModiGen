
We have a container, and we want to keep the height of the water (`waterLevel`) in the container at a target height (`target_height`) of 1 meter. The water height (`waterLevel`) in the pool starts at 0.2m, and the pool will leak 0.1m per unit time. Use proportional integral control to implement the above control system (defining height error as `error`, and splitting the logic into proportional control `Pcontrol`, which handles the leak, and integral control `Icontrol`) and simulate the change of the water level over time, where the proportional coefficient (`Kp`) is 1 and the integral coefficient (`Ki`) is 0.5. Please help me implement a modelica model named `PIController`.

//modelica
```model PIController
  parameter Real waterLevel(min=0) = 0.2;
  parameter Real target_height = 1;
  parameter Real Kp = 1;
  parameter Real Ki = 0.5;
  parameter Real leakRate = 0.1;
  Real error;
  Real Pcontrol;
  Real Icontrol;
  Real output;
initial equation
  error = target_height - waterLevel;
equation
  Pcontrol = Kp * error;
  Icontrol = Ki * integral(error, 0);
  output = Pcontrol + Icontrol;
  der(waterLevel) = output - leakRate;
end PIController;
```

It then calculates the proportional and integral control outputs based on the error and integrates the error over time to calculate the integral control output. The total control output is the sum of the proportional and integral control outputs. The change in water level is determined by the control output minus the leak rate.

Please note that the integral term is calculated using the `integral` function, which calculates the integral of the input over time. The initial condition for the integral term is 0, which means that the integral starts from the initial value of the error.

The `der` function is used to calculate the derivative of the water level with respect to time. The derivative is determined by the control output minus the leak rate. This equation represents the differential equation that describes the change in water level over time.
