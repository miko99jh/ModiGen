
Suppose we have a hydraulic system consisting of two oil storage tanks and a cylindrical connecting pipe. It is known that the density of oil (`rho`) in the tank is 0.2; The initial height of the liquid in the left tank (`H1`) is 0.4 and the cross-sectional area (`A1`) is 1; The initial height of the liquid in the right tank (`H2`) is 0.6 and the cross-sectional area (`A2`) is 2. The connection pipe has a length (`L`) of 0.1 and a diameter (`D`) of 0.2, and the dynamic viscosity (`mu`) of the oil is 2e-3. The model also calculates a flow coefficient (`c`) based on the pipe dimensions and viscosity. Please help me implement the modelica model named `TwoTanks` of this system, including equations for pressures (`P1`, `P2`), flow rate (`Q`), and the conservation of mass to determine the rate of change of the heights.

//modelica
```model TwoTanks
  parameter Real rho = 0.2; // density of oil
  parameter Real H1 = 0.4; // initial height of oil in the left tank
  parameter Real A1 = 1; // cross-sectional area of the left tank
  parameter Real H2 = 0.6; // initial height of oil in the right tank
  parameter Real A2 = 2; // cross-sectional area of the right tank
  parameter Real L = 0.1; // length of the pipe
  parameter Real D = 0.2; // diameter of the pipe
  parameter Real mu = 2e-3; // dynamic viscosity of the oil
  parameter Real c = (pi*D^4)/(128*L*mu); // flow coefficient
  Real P1; // pressure in the left tank
  Real P2; // pressure in the right tank
  Real Q; // flow rate
  Real dh1_dt; // rate of change of the height of oil in the left tank
  Real dh2_dt; // rate of change of the height of oil in the right tank
equation 
  // equations for pressures
  P1 = rho*g*H1;
  P2 = rho*g*H2;
  // equation for flow rate
  Q = c*(P1 - P2);
  // conservation of mass
  dh1_dt = -Q/A1;
  dh2_dt = Q/A2;
end TwoTanks;
```

This model should be able to calculate the pressures (`P1`, `P2`), flow rate (`Q`), and the rate of change of the heights (`dh1_dt`, `dh2_dt`) of the oil in the tanks.

Note: The gravitational acceleration (`g`) is assumed to be 9.81 m/s^2.
