
Implement a Modelica function named `SphereProperties` to calculate the surface area (`surface`) and volume (`volume`) of a sphere based on its radius (`radius`). The function should use an algorithm block to implement the calculation logic and utilize the pi constant from the standard Modelica library (Modelica.Constants.pi) for the formulas.

//modelica
```function SphereProperties
  input Real radius;
  output Real surface;
  output Real volume;
algorithm
  surface := 4 * Modelica.Constants.pi * radius^2;
  volume := 4/3 * Modelica.Constants.pi * radius^3;
end SphereProperties;```

This function takes a single input, `radius`, which represents the radius of the sphere. It calculates the surface area and volume of the sphere using the formulas for a sphere, which are `surface = 4 * pi * radius^2` and `volume = 4/3 * pi * radius^3`, respectively. The results are returned as the outputs `surface` and `volume`.

The algorithm block is used to implement the calculation logic. It uses the pi constant from the Modelica.Constants package for the formulas. The `:=` operator is used to assign the calculated values to the outputs.

The function can be used in a Modelica model to calculate the surface area and volume of a sphere based on its radius.

  Real surface "Surface area of the sphere";
  Real volume "Volume of the sphere";
initial equation

end SphereModel;```
