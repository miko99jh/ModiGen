1. Introduction to Python
=========================

Python is a high-level, interpreted programming language that was created by Guido van Rossum in 1991. It is known for its simplicity and readability, making it a great language for beginners. Python is used in a wide range of applications, from web development to data analysis, and is supported by a large and active community of users and developers.

Python's design philosophy emphasizes code readability with its use of significant indentation. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.

Python Features
----------------

Here are some of the key features of Python:

1. **Easy-to-learn**: Python has simpler syntax comparing to other languages. This simplicity allows the programmers to develop applications much faster.

2. **Interpreted**: Python is an interpreted language, which means that it converts the source code into an intermediate language, which is then translated into machine language that has to be executed.

3. **Interactive**: Python supports interactive mode which means that the user can interact with the interpreter directly.

4. **Portable**: Python is portable, i.e., it can be transferred to other platforms.

5. **Extendable**: Python can be extended from C, C++, or Java.

6. **Object-Oriented**: Python supports Object-Oriented style or technique of programming that encapsulates data and functions into separate objects.

7. **GUI Programming**: Python supports GUI (Graphical User Interface) Programming.

8. **Scalable**: Python provides a better structure and support for large-scale applications.

Python Syntax
--------------

Python syntax is very clean and easy to understand. It is designed to be highly readable, which means that it uses English keywords frequently where other languages use punctuation. Python's simplicity lets you learn more about the subject in less time.

Here is a simple Python program:

.. code-block:: python

   print("Hello, World!")

Python Variables

Python variables are containers for storing data values. Variables are created the moment you first assign a value to it.

Python supports four different numerical types:

Python Strings
----------------

Strings in python are identified as a contiguous set of characters represented in the quotation marks. Python allows for either pairs of single or double quotes.

Python Lists
--------------

Lists are used to store multiple items in a single variable. Lists are one of 4 built-in data types in Python used to store collections of data, the other 3 are Tuple, Set, and Dictionary, all with different qualities and usage.

Python Tuples
----------------

A tuple is a collection of objects which ordered and immutable. Tuples are sequences, just like lists. The differences between tuples and lists are, the tuples cannot be changed unlike lists and tuples use parentheses, whereas lists use square brackets.

Python Dictionaries
--------------------

Python's dictionaries are kind of hash-table type. They work like associative arrays or hashes which are almost similar to arrays except that you can use strings as indices.

Python If...Else
------------------

Python programming language assumes any non-zero and non-null values as true, and if it is either zero or null, then it is assumed as false value.

A while loop statement in Python programming language repeatedly executes a target statement as long as a given condition is true.
