
Generate a Modelica model named `FirstOrderExperiment` that defines first order equation with experiment conditions for a simple system. The model should include a real variable 'x' and initialize 'x=2'. This variable should dynamically change based on the differential equation defined, aiming for the value of 'x' to converge towards 1.0.

//modelica
```model FirstOrderExperiment
  parameter Real x(start=2) "State variable";
  parameter Real alpha(start=0.1) "Rate of convergence";
equation
  der(x) = -alpha*x;
  when x<1.0 then
    stop();
  end when;
end FirstOrderExperiment;```

The `end when` statement is used to end the `when` statement.

The `end FirstOrderExperiment` statement is used to end the model.
