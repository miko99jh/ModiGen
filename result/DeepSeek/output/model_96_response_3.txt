
Simulate the flow of fluid through a pipe. The known pressure at both ends of the pipeline are `P1` = 1080 and `P2` = 900. The pipe length (`L`) and inner diameter (`D`) are 1 and 0.05, respectively. The viscosity of the fluid (`mu`) is 2e-3. Please help me realize the Modelica model named `Pipe` corresponding to the above system to calculate a flow coefficient (`c`), the flow in the pipeline (`Q`), and the velocity (`v`). The model should use the Hagen-Poiseuille relationship where the flow (`Q`) is determined by the pressure difference and the coefficient (`c`), and the velocity (`v`) is derived from the flow and the pipe's cross-sectional area.

//modelica
```model Pipe
  parameter Real P1 = 1080 "Pressure at the start of the pipe";
  parameter Real P2 = 900 "Pressure at the end of the pipe";
  parameter Real L = 1 "Length of the pipe";
  parameter Real D = 0.05 "Inner diameter of the pipe";
  parameter Real mu = 2e-3 "Viscosity of the fluid";
  Real c "Flow coefficient";
  Real Q "Flow in the pipeline";
  Real v "Velocity";
equation 
  // equation for the flow coefficient
  c = (8*mu*L)/(pow(D,4));
  // equation for the flow in the pipeline
  Q = c*(P1 - P2);
  // equation for the velocity
  v = Q/(pi()*pow(D,2)/4);
end Pipe;
```

The model is not correct because the velocity equation is not correct. The velocity should be calculated as `v = Q/(pi()*D^2/4)`. Please help me correct the model.
