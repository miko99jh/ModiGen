
Generate a Modelica model named `Damper` for a rotational damper. The model should include a single parameter 'd' of type 'Modelica.SIunits.RotationalDampingConstant' representing the rotational damping constant with units of Newton meters per radian per second. The model should extend 'ModelicaByExample.Components.Rotational.Interfaces.Compliant' and implement an equation for the damping relationship, where the torque 'tau' equals 'd' multiplied by the derivative of 'phi_rel'. This model simulates the damping effect in rotational systems, mitigating oscillations and stabilizing motion.

//modelica
```model Damper
  extends ModelicaByExample.Components.Rotational.Interfaces.Compliant;
  parameter Modelica.SIunits.RotationalDampingConstant d;
equation
  tau = d * der(phi_rel);
end Damper;
```

This model is a simple representation of a rotational damper that can be used in rotational systems. The torque 'tau' is directly proportional to the rate of change of the relative angular position 'phi_rel' and the damping constant 'd'. This model can be used to simulate the damping effect in rotational systems, mitigating oscillations and stabilizing motion.

Please note that this model is a simplified representation and may not accurately represent the behavior of a real-world rotational damper. For more accurate simulations, a more complex model that takes into account the physical characteristics of the damper may be required.
