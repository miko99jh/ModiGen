The `Feedback` model is a simple block that computes the feedback term in a control system. It takes two real inputs, `u1` and `u2`, and computes the difference between them. The result is then outputted as `y`. This model can be used in a feedback loop to control a system.

## Model Parameters

There are no parameters for the `Feedback` model.

## Model Equations

The equation for the `Feedback` model is as follows:

```
y = u1.y - u2.y
```

This equation computes the difference between the outputs of `u1` and `u2`, and outputs the result as `y`.
