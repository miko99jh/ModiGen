
Implement the Modelica function named `Rt_T` for solving the temperature characteristics of platinum resistance. The input is the current temperature (`T`) and the output is the resistance value (`Rt`) of the platinum resistor. It is known that the reference resistance at 0°C is `R0` = 100 ohms. The function should use an `algorithm` block and define `R0` along with constants `A` = 3.9083*10^(-3), `B` = -5.775*10^(-7), and `C` = -4.183*10^(-12) as protected internal variables. When (`T` >= 0) and (`T` < 850), the resistance is calculated as `Rt` = R0 * (1 + A*T + B*T^2); When (`T` > -200) and (`T` < 0), the resistance is calculated as `Rt` = R0 * (1 + A*T + B*T^2 + C*T^3 * (T - 100)).

//modelica
```function Rt_T
  input Real T "Temperature";
  output Real Rt "Resistance at temperature T";
protected 
  constant Real R0 = 100 "Reference resistance at 0 degree Celsius";
  constant Real A = 3.9083*10^(-3) "Coefficient A";
  constant Real B = -5.775*10^(-7) "Coefficient B";
  constant Real C = -4.183*10^(-12) "Coefficient C";
algorithm 
  if (T >= 0) and (T < 850) then
    Rt := R0 * (1 + A*T + B*T^2);
  elseif (T > -200) and (T < 0) then
    Rt := R0 * (1 + A*T + B*T^2 + C*T^3 * (T - 100));
  else
    Rt := R0;
  end if;
end Rt_T;```

This function calculates the resistance of a platinum resistor at a given temperature. The resistance decreases as the temperature increases, and the rate of decrease increases as the temperature increases further. The function uses a piecewise linear approximation to model this behavior. The resistance is constant at 100 ohms between 0 and 850 degrees Celsius, and decreases more rapidly above 850 degrees Celsius. It also decreases more rapidly below -200 degrees Celsius.

Note that this function does not check whether the input temperature is within the valid range for platinum resistors (-200 to 850 degrees Celsius). It is the responsibility of the caller to ensure that the input temperature is within this range.
