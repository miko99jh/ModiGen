
We have a container, and we want to keep the height of the water (`waterLevel`) in the container at a target height (`target_height`) of 1 meter. The water height (`waterLevel`) in the pool starts at 0.2m, and the pool will leak 0.1m per unit time. Use proportional integral control to implement the above control system (defining height error as `error`, and splitting the logic into proportional control `Pcontrol`, which handles the leak, and integral control `Icontrol`) and simulate the change of the water level over time, where the proportional coefficient (`Kp`) is 1 and the integral coefficient (`Ki`) is 0.5. Please help me implement a modelica model named `PIController`.

//modelica
```model PIController
  parameter Real Kp = 1 "Proportional coefficient";
  parameter Real Ki = 0.5 "Integral coefficient";
  parameter Real target_height = 1 "Target height";
  parameter Real initial_waterLevel = 0.2 "Initial water level";
  parameter Real leak_rate = 0.1 "Leak rate";
  Real waterLevel(start = initial_waterLevel) "Water level";
  Real error "Error";
  Real Pcontrol "Proportional control";
  Real Icontrol "Integral control";
  Real integral "Integral of error";
equation
  // Define the error
  error = target_height - waterLevel;
  // Implement the proportional control
  Pcontrol = Kp * error;
  // Implement the integral control
  integral = integral + Ki * error * timeStep;
  Icontrol = integral;
  // Update the water level
  der(waterLevel) = Pcontrol + Icontrol - leak_rate;
end PIController;
```

The target water level is defined by the parameter `target_height`. The initial water level is defined by the parameter `initial_waterLevel`. The leak rate is defined by the parameter `leak_rate`. The water level is defined by the state variable `waterLevel`. The error is defined as the difference between the target water level and the current water level. The proportional control is defined as the product of the proportional gain and the error. The integral control is defined as the integral of the error over time, multiplied by the integral gain. The water level is updated in each time step as the sum of the proportional control, the integral control, and the water level leakage, minus the water level leakage in the previous time step.

The `timeStep` is a built-in variable in Modelica that represents the time step size of the simulation. It is used here to ensure that the integral is calculated over the correct time interval.
