
Generate a Modelica model named `FirstOrderExperiment` that defines first order equation with experiment conditions for a simple system. The model should include a real variable 'x' and initialize 'x=2'. This variable should dynamically change based on the differential equation defined, aiming for the value of 'x' to converge towards 1.0.

//modelica
```model FirstOrderExperiment
  parameter Real x(start=2) "Variable x";
  parameter Real k = 0.1 "Rate constant";
equation
  der(x) = -k*x;
  annotation(experiment(StartTime=0, StopTime=100, Tolerance=1e-6, Interval=10));
end FirstOrderExperiment;
```

This model defines a first order differential equation where 'x' decreases over time due to the negative feedback term '-k*x'. The parameter 'k' is the rate constant that determines the rate of change of 'x'. The experiment annotation is used to simulate the model from 0 to 100 seconds with a tolerance of 1e-6 and a simulation interval of 10 seconds. The initial value of 'x' is set to 2.
