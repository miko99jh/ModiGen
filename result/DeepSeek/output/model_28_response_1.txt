
Generate a Modelica block named `ConjunctiveClock` with the following for a logical clock that ticks when all input clocks have ticked at least once, then resets for the next cycle. This block extends `Modelica.Clocked.ClockSignals.Clocks.Logical.PartialLogicalClock`. It also configure the block to perform a logical conjunction (AND) by redeclaring the internal 'combinator' component with the appropriate Boolean logic block from the Modelica Standard Library in `PartialLogicalClock`.

//modelica
```block ConjunctiveClock extends Modelica.Clocked.ClockSignals.Clocks.Logical.PartialLogicalClock(combinator = Modelica.Blocks.Logical.And);```

The `ConjunctiveClock` block should have an input connector `in` and an output connector `out` that are of type `ClockSignal`.

//modelica
```connector in input ClockSignal;
connector out output ClockSignal;```

//modelica
```parameter Integer n=2 "Number of input clocks";```

The `ConjunctiveClock` block should have an array `inputClocks` of size `n` that represents the input clocks.

//modelica
```input ClockSignal inputClocks[n](start=false) "Input clocks";```

The `ConjunctiveClock` block should have a `when` clause that triggers the `combinator` component when all input clocks have ticked at least once.

//modelica
```when all inputClocks.tick then combinator.trigger;```

The `ConjunctiveClock` block should have a `reset` clause that resets the `combinator` component for the next cycle.

//modelica
```reset when not all inputClocks.tick;```

The `ConjunctiveClock` block should connect the `in` connector to the `inputClocks` array and the `out` connector to the `combinator.out` connector.

//modelica

connect(combinator.out, out);```
