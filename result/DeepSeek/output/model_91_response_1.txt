
We have a container, and we want to keep the height of the water (`waterLevel`) in the container at a target height (`target_height`) of 1 meter. The water height (`waterLevel`) in the pool starts at 0.2m, and the pool will leak 0.1m per unit time. Use proportional integral control to implement the above control system (defining height error as `error`, and splitting the logic into proportional control `Pcontrol`, which handles the leak, and integral control `Icontrol`) and simulate the change of the water level over time, where the proportional coefficient (`Kp`) is 1 and the integral coefficient (`Ki`) is 0.5. Please help me implement a modelica model named `PIController`.

//modelica
```model PIController
  parameter Real target_height = 1;
  parameter Real Kp = 1;
  parameter Real Ki = 0.5;
  parameter Real leakRate = 0.1;
  parameter Real initial_waterLevel = 0.2;
  Real waterLevel(start = initial_waterLevel);
  Real error(start = target_height - initial_waterLevel);
  Real Pcontrol(start = Kp * error);
  Real Icontrol(start = 0);
equation
  // Define the differential equation for the water level
  der(waterLevel) = Pcontrol + Icontrol - leakRate;
  // Define the integral equation for the integral control
  Icontrol = Ki * integral(error);
  // Define the error as the difference between the target height and the current water level
  error = target_height - waterLevel;
end PIController;
```

This model should simulate the water level in the container over time. The water level starts at 0.2m and leaks at a rate of 0.1m/s. The proportional control attempts to keep the water level at 1m, and the integral control attempts to maintain this over time. The proportional coefficient is 1 and the integral coefficient is 0.5.

Please note that the integral control term is calculated based on the error, which is the difference between the target height and the current water level. The integral control attempts to accumulate the error over time and apply it as a correction to the proportional control.

The differential equation for the water level is defined as the sum of the proportional control, the integral control, and the leakage rate. The integral equation for the integral control is defined as the integral of the error.

The simulation should show how the water level changes over time in response to the control system.
