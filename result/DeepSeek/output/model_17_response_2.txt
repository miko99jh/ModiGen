
Generate a Modelica model named `Add` that adds two integer inputs 'u1' and 'u2' and outputs their sum 'y'. The model should extend the `IndustrialControlSystems.MathOperations.IntegerType.Interfaces.IntSimpleOperation`. The input 'u' and output 'y' exist in IntSimpleOperation and does not need to be redeclared. It should handle the addition through a conditional operation based on the sampling time Ts. If Ts is greater than zero, the output is computed at each sample point; otherwise, the output is the direct sum of the two inputs.

//modelica
```model Add
  extends IndustrialControlSystems.MathOperations.IntegerType.Interfaces.IntSimpleOperation;
  parameter Integer u1;
  parameter Integer u2;
  output Integer y;
equation
  if Ts > 0 then
    der(y) = (u1 + u2);
  else
    y = u1 + u2;
  end if;
end Add;
```

This model extends the `IntSimpleOperation` interface and defines two parameters `u1` and `u2` for the inputs. The output `y` is calculated based on the sampling time `Ts`. If `Ts` is greater than zero, the output is computed at each sample point by taking the derivative of `y` with respect to time and setting it equal to the sum of `u1` and `u2`. If `Ts` is zero or negative, the output is simply the sum of `u1` and `u2`.

Note: This model assumes that the inputs `u1` and `u2` are known constants and that the output `y` is a state variable if `Ts` is greater than zero. If `Ts` is zero or negative, the output `y` is not a state variable and can be computed directly.

Please replace `IndustrialControlSystems.MathOperations.IntegerType.Interfaces.IntSimpleOperation` with the correct path to the interface in your Modelica library if it's different.
