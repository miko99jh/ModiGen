
Please use modelica to help me implement a model named `BouncingBall`: This is a bouncing ball model with a radius of 0.1m, an initial height of 1m and an initial velocity of 0 m/s. Affected by gravity, the ball moves in free fall and bounces upward at the moment it hits the ground. The recovery coefficient during the collision is 0.9.

//modelica
```model BouncingBall
  parameter Real radius = 0.1 "Ball radius";
  parameter Real initialHeight = 1 "Initial height";
  parameter Real initialVelocity = 0 "Initial velocity";
  parameter Real g = 9.81 "Gravity acceleration";
  parameter Real recoveryCoefficient = 0.9 "Recovery coefficient";

  Real height(start=initialHeight) "Height";
  Real velocity(start=initialVelocity) "Velocity";
  Real acceleration "Acceleration";

equation
  acceleration = -g; // Free fall
  der(velocity) = acceleration; // Velocity derivative
  der(height) = velocity; // Height derivative

  // Bounce equation
  when height <= 0 then
    velocity = -recoveryCoefficient * velocity;
    height = 0;
  end when;
end BouncingBall;
```

This model is a simple representation of a bouncing ball. It uses the equations of motion for a freely falling body and the law of conservation of energy to model the bouncing behavior. The model assumes that the ball loses energy during the bounce due to friction with the ground, which is modeled by the recovery coefficient.

Please note that this model does not take into account the air resistance, which would significantly affect the motion of a real bouncing ball. The air resistance can be included in the model by adding a drag force that opposes the motion of the ball.

Also, the model assumes that the ball hits the ground with a perfectly elastic collision, which is not realistic. However, this assumption simplifies the model and allows us to derive the equations of motion for a bouncing ball.
