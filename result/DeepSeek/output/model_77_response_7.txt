
Generate a Modelica function named `convertAlpha` that converts a linear temperature coefficient (alpha) from one temperature to another. The function should include the following components and details:
1. Components:
-  The function should extend from Modelica.Icons.Function. 
-  Input for the linear temperature coefficient at temperature 1: input Modelica.Units.SI.LinearTemperatureCoefficient alpha1. 
-  Input for temperature 2: input Modelica.Units.SI.Temperature T2. 
-  Input for temperature 1 with a default value: input Modelica.Units.SI.Temperature T1=293.15. 
-  Output for the linear temperature coefficient at temperature 2: output Modelica.Units.SI.LinearTemperatureCoefficient alpha2.
2. Algorithm:
 The algorithm should calculate the new temperature coefficient 'alpha2' by shifting the reference baseline from 'T1' to 'T2'.
The conversion should account for the reciprocal nature of linear coefficients when changing reference points, such that the ratio of the coefficients is adjusted by the linear factor defined by the temperature difference and the original coefficient.

Please ensure that the function is syntactically correct and adheres to the Modelica language standard.

//modelica
```function convertAlpha
  extends Modelica.Icons.Function;
  input Modelica.Units.SI.LinearTemperatureCoefficient alpha1;
  input Modelica.Units.SI.Temperature T2;
  input Modelica.Units.SI.Temperature T1=293.15;
  output Modelica.Units.SI.LinearTemperatureCoefficient alpha2;
algorithm

end convertAlpha;
```
