
Define an array (`data`) of length (`n = 5`) and initialize it with the values {1, 2, 3, 4, 5}. Solve for the absolute values (`abs1`) of the array elements, the minimum value (`min1`), the maximum value (`max1`), and the sum (`sum1`) of all elements. Please help me generate a Modelica model named `ArrayExample` to do this using an equation block.

//modelica
```model ArrayExample
  parameter Integer n = 5;
  Real data[n] = {1, 2, 3, 4, 5};
  Real abs1[n];
  Real min1;
  Real max1;
  Real sum1;
equation
  for i in 1:n loop
    abs1[i] = abs(data[i]);
  end for;
  min1 = min(data);
  max1 = max(data);
  sum1 = sum(data);
end ArrayExample;
```

This Modelica model named `ArrayExample` has five parameters: `n` (the size of the array), `data` (the array of values), `abs1` (the array of absolute values), `min1` (the minimum value), `max1` (the maximum value), and `sum1` (the sum of all values). The equations in the equation block calculate these values.

The `for` loop iterates over each element in the array and assigns the absolute value of the element to the corresponding element in the `abs1` array. The `min` and `max` functions find the minimum and maximum values in the `data` array, respectively. The `sum` function calculates the sum of all values in the `data` array.

Please note that the `min`, `max`, and `sum` functions are built-in Modelica functions that operate on arrays.
