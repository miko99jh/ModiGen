
Generate a Modelica model named `Damper` for a rotational damper. The model should include a single parameter 'd' of type 'Modelica.SIunits.RotationalDampingConstant' representing the rotational damping constant with units of Newton meters per radian per second. The model should extend 'ModelicaByExample.Components.Rotational.Interfaces.Compliant' and implement an equation for the damping relationship, where the torque 'tau' equals 'd' multiplied by the derivative of 'phi_rel'. This model simulates the damping effect in rotational systems, mitigating oscillations and stabilizing motion.

//modelica
```model Damper
  extends ModelicaByExample.Components.Rotational.Interfaces.Compliant;
  parameter Modelica.SIunits.RotationalDampingConstant d;
equation 
  tau = d * der(phi_rel);
end Damper;
```

This model is a simple rotational damper that can be used in rotational systems. It extends the 'Compliant' interface from the 'Rotational' package in 'ModelicaByExample.Components.Rotational.Interfaces' and implements the damping relationship. The torque 'tau' is calculated as 'd' multiplied by the derivative of the relative angle 'phi_rel', which represents the damping effect. The parameter 'd' is of type 'Modelica.SIunits.RotationalDampingConstant', which ensures that it has the correct units for a rotational damping constant.

Please note that this model assumes a linear relationship between the torque and the angular velocity, which may not be accurate for all types of dampers. For more complex dampers, a more sophisticated model may be required.
