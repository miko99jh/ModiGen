
Generate a Modelica model named `IdealOpAmp3Pin` for an ideal operational amplifier (norator-nullator pair) with 3 pins. The model should include the following components:
1. `Modelica.Electrical.Analog.Interfaces.PositivePin in_p` as the positive input pin.
2. `Modelica.Electrical.Analog.Interfaces.NegativePin in_n` as the negative input pin.
3. `Modelica.Electrical.Analog.Interfaces.PositivePin out` as the output pin.

In the equation section, define the following behavior:
- The voltage at the positive input pin `in_p.v` should be equal to the voltage at the negative input pin `in_n.v`.
- The current at both the positive input pin `in_p.i` and the negative input pin `in_n.i` should be zero.

Provide the necessary equations to represent the ideal operational amplifier's behavior, with balanced input voltages and zero input currents.

//modelica
```model IdealOpAmp3Pin
  extends Modelica.Electrical.Analog.Interfaces.OnePort;
  Modelica.Electrical.Analog.Interfaces.PositivePin in_p;
  Modelica.Electrical.Analog.Interfaces.NegativePin in_n;
equation 
  in_p.v = in_n.v; // Voltage at positive input pin is equal to the voltage at negative input pin
  in_p.i = in_n.i = 0; // Current at both positive input pin and negative input pin is zero
end IdealOpAmp3Pin;
```

This model represents an ideal operational amplifier with 3 pins. The voltage at the positive input pin is equal to the voltage at the negative input pin, and the current at both the positive input pin and the negative input pin is zero, thus satisfying the behavior of an ideal operational amplifier.
