
Implement a Modelica function named `CircleProperties` to calculate the circumference (`circumference`) and area (`area`) of a circle based on its radius (`radius`). The function should include a protected section to define the diameter (`diameter`) and use an algorithm block to implement the logic for both outputs. Assume the value of pi is 3.14159 for the calculations.

//modelica
```function CircleProperties
  input Real radius;
  output Real circumference;
  output Real area;
protected 
  Real diameter;
algorithm 
  diameter := 2 * radius;
  circumference := 3.14159 * diameter;
  area := 3.14159 * radius * radius;
end CircleProperties;```

This function takes a radius as input and calculates the diameter by multiplying the radius by 2. It then calculates the circumference by multiplying pi (3.14159) by the diameter, and the area by multiplying pi by the square of the radius. The results are then returned as the output values for circumference and area.

Note: This function assumes that the radius is a positive value. If the radius is negative, the results will be incorrect.

You can use this function in your Modelica models by calling it and providing the radius as an argument. The function will return the circumference and area of the circle.

For example:

```modelica
CircleProperties(radius=5);
```
