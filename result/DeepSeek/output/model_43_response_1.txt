
Generate a Modelica block named 'Add' that outputs the sum of two complex input signals.

# Model Functionality:
- The block computes the sum of two complex input signals, each multiplied by a respective gain factor (k1 and k2). 
- The parameters 'k1' and 'k2' are complex parameters with default values of Complex(1, 0).

# Inputs and Outputs:
- **Inputs**:
- `u1Internal`: Complex input signal 1. 
- `u2Internal`: Complex input signal 2. 
- **Outputs**:
- `y`: Complex output signal, representing the sum of the two weighted inputs. 

# Inheritance and Interfaces:
- The block extends from `Modelica.ComplexBlocks.Interfaces.ComplexSI2SO`, which indicates that it follows a two-input, single-output interface with complex signal types. 

Ensure the model adheres to the specified input-output relationships and complex arithmetic operations.

//modelica
```block Add
model Add "Add complex signals"
  extends Modelica.ComplexBlocks.Interfaces.ComplexSI2SO;
  parameter Modelica.ComplexMath.Complex ck1=Complex(1, 0) "Gain factor for u1";
  parameter Modelica.ComplexMath.Complex ck2=Complex(1, 0) "Gain factor for u2";
  Modelica.ComplexMath.ComplexInput ci1(redeclare package Modelica.ComplexMath model=Modelica.ComplexMath.ComplexInput);
  Modelica.ComplexMath.ComplexInput ci2(redeclare package Modelica.ComplexMath model=Modelica.ComplexMath.ComplexInput);
  Modelica.ComplexMath.ComplexOutput co(redeclare package Modelica.ComplexMath model=Modelica.ComplexMath.ComplexOutput);
equation 
  connect(ci1, u1Internal);
  connect(ci2, u2Internal);
  connect(ci1.output, co.input);
  connect(ci2.output, co.input);
  y = ck1*ci1.output + ck2*ci2.output;
end Add;
```

This Modelica block can be used in a complex signal processing system to add two complex signals after scaling them by the gain factors.
