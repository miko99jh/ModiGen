
We have a container, and we want to keep the height of the water (`waterLevel`) in the container at a target height (`target_height`) of 1 meter. The water height (`waterLevel`) in the pool starts at 0.2m, and the pool will leak 0.1m per unit time. Use proportional integral control to implement the above control system (defining height error as `error`, and splitting the logic into proportional control `Pcontrol`, which handles the leak, and integral control `Icontrol`) and simulate the change of the water level over time, where the proportional coefficient (`Kp`) is 1 and the integral coefficient (`Ki`) is 0.5. Please help me implement a modelica model named `PIController`.

//modelica
```model PIController
  parameter Real target_height = 1 "Target height of water in container";
  parameter Real initial_waterLevel = 0.2 "Initial height of water in container";
  parameter Real leakRate = 0.1 "Rate at which water leaks out of container";
  parameter Real Kp = 1 "Proportional coefficient";
  parameter Real Ki = 0.5 "Integral coefficient";
  Real waterLevel(start = initial_waterLevel) "Height of water in container";
  Real error(start = target_height - initial_waterLevel) "Height error";
  Real Pcontrol "Proportional control";
  Real Icontrol "Integral control";
equation 
  // Define the proportional control
  Pcontrol = Kp * error;

  // Define the integral control
  Icontrol = Ki * (integral(error, 0) + error);

  // Define the change in water level
  der(waterLevel) = Pcontrol + Icontrol - leakRate;

  // Define the error
  error = target_height - waterLevel;
end PIController;
```

This model is a simple implementation of a PI controller for a system where the water level in a container is controlled to a target height. The proportional control (`Pcontrol`) is used to counteract the leak rate, while the integral control (`Icontrol`) is used to maintain the target height. The change in water level (`der(waterLevel)`) is calculated by adding the proportional and integral controls, and subtracting the leak rate. The error is calculated as the difference between the target height and the current water level.

Please note that this model assumes that the leak rate is constant and does not change over time. If the leak rate changes over time, you would need to modify the model to account for this.

Also, this model does not include any initial or boundary conditions. These would need to be specified depending on the specifics of the system you are modeling.
