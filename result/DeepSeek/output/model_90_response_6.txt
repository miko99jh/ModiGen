
We have a reservoir, and we want to keep the height of the water (`waterLevel`) in the reservoir at a target height (`target_height`) of 1m. The water height (`waterLevel`) in the pool is 0.2m at the beginning. Please use proportional control to implement the above control system (defining the altitude error as `error` and the control signal as `control`) and simulate the change of the water level in the pool over time with a proportional factor (`Kp`) of 1. Please help me implement a modelica model named `PController`.

//modelica
```model PController
  parameter Real target_height = 1 "Target height of water in reservoir";
  parameter Real Kp = 1 "Proportional control factor";
  Real waterLevel(start = 0.2) "Water level in the pool";
  Real error "Error between target and actual water level";
  Real control "Control signal";
equation 
  // Define the error
  error = target_height - waterLevel;

  // Define the control signal
  control = Kp * error;

  // Define the change of water level over time
  der(waterLevel) = control;
end PController;
```

This model will simulate the change of the water level in the pool over time based on the proportional control signal. The `der` function is used to define the derivative of `waterLevel` with respect to time. The `start` attribute is used to define the initial value of `waterLevel`.

Please note that this model assumes that the control signal directly affects the change of the water level. In reality, the control signal may need to be integrated over time to affect the water level. If this is the case, you may need to modify the model to include an integration term.

Also, this model does not include any limits on the water level. In a real-world application, you may want to include logic to prevent the water level from going below a certain value or above another value.

Finally, this model does not include any dynamics in the system. In a real-world application, you may want to include a time constant to represent the time it takes for the system to respond to the control signal.
