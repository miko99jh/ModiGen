
Create a Modelica model named `NewtonCooling` for Newton's law of cooling. Key parameters include ambient temperature (Temperature T_inf=300.0), initial temperature (Temperature T0=280.0), convective cooling coefficient (ConvectionCoefficient h=0.7), surface area (Area A=1.0), mass of thermal capacitance (Mass m=0.1), and specific heat capacity (SpecificHeat c_p=1.2). The output should be the temperature change over time, governed by the formula m*c_p*der(T) = h*A*(T_inf-T), depicting the cooling process. This model is commonly used in thermal analysis to understand the cooling dynamics of objects exposed to an environment with a different temperature.

//modelica
```model NewtonCooling
  parameter Real T_inf = 300.0 "Ambient temperature";
  parameter Real T0 = 280.0 "Initial temperature";
  parameter Real h = 0.7 "Convection coefficient";
  parameter Real A = 1.0 "Surface area";
  parameter Real m = 0.1 "Mass";
  parameter Real c_p = 1.2 "Specific heat capacity";

  Real T(start=T0) "Temperature";

equation
  der(T) = (h*A*(T_inf-T) - m*c_p*der(T)) / (m*c_p);
end NewtonCooling;
```

This model uses Newton's law of cooling, which states that the rate of heat loss of a body is directly proportional to the difference in temperatures between the body and its surroundings. The model assumes that the only heat transfer mechanism is convection, and that the heat capacity of the body is constant.

The equation `der(T) = (h*A*(T_inf-T) - m*c_p*der(T)) / (m*c_p)` is derived from Newton's law of cooling and represents the rate of change of temperature with time. It is rearranged to solve for `der(T)` to make the model solvable by a numerical method.

The initial temperature `T0` is set to 280.0 degrees Celsius, and the ambient temperature `T_inf` is set to 300.0 degrees Celsius. The convective cooling coefficient `h` is set to 0.7, the surface area `A` is set to 1.0 square meter, the mass `m` is set to 0.1 kilogram, and the specific heat capacity `c_p` is set to 1.2 Joules per kilogram per degree Celsius.

The temperature `T` is a state variable that changes over time according to the equation. The derivative of `T` with respect to time is the rate of change of temperature.

This model can be used to simulate the cooling of an object in a thermal analysis application.
