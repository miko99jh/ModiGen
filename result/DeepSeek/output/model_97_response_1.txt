
Please simulate the temperature change of a platinum resistance sensor in a medium. It is known that the temperature of the measured medium (`Tg`) is stable at 100°C. The volume of the sensor (`V`) is 1.15*10^(-9), the comprehensive specific heat (`c`) is 456, the sensor surface area (`Ar`) is 5.31*10^(-6), the convective heat transfer coefficient (`alpha`) is 800, and the comprehensive density (`pho`) is 7980. The initial temperature of the sensor (`Tj`) is 25 °C. Please help me realize the Modelica model named `RTD_Pt100` where the medium temperature (`Tg`) is defined as an **input** and the sensor temperature (`Tj`) is defined as an **output**. The model should calculate the temperature change of the sensor over time based on the heat balance between the sensor and the medium.

//modelica
```model RTD_Pt100
  parameter Real V = 1.15*10^(-9); // volume of the sensor
  parameter Real c = 456; // comprehensive specific heat
  parameter Real Ar = 5.31*10^(-6); // sensor surface area
  parameter Real alpha = 800; // convective heat transfer coefficient
  parameter Real pho = 7980; // comprehensive density
  Real Tj(start = 25); // sensor temperature
  input Real Tg; // medium temperature
  Real Q_conv; // convective heat transfer
  Real Q_rad; // radiative heat transfer
  Real Q_sens; // sensible heat
  Real m; // mass of the sensor
  Real A; // sensor surface area
  Real R; // resistance of the sensor
  Real V_j; // voltage across the sensor
  Real Tj_prev; // previous sensor temperature
  equation
    // mass of the sensor
    m = V*pho;
    // sensor surface area
    A = 2*Ar;
    // resistance of the sensor
    R = 100; // this is a dummy value, replace it with the actual resistance calculation
    // voltage across the sensor
    V_j = R*m*(Tj - Tg)/(Tj - Tj_prev);
    // heat balance equation
    Q_conv = alpha*A*(Tj - Tg);
    Q_rad = 0; // this is a dummy value, replace it with the actual radiative heat transfer calculation
    Q_sens = m*c*(Tj - Tj_prev)/V;
    // differential equation for sensor temperature
    der(Tj) = (Q_conv + Q_rad + Q_sens)/m/c;
    // update previous sensor temperature
    Tj_prev = Tj;
end RTD_Pt100;
```

Please note that this model is a simplified version and does not include all the factors that may affect the temperature change of the sensor, such as the sensor resistance and the radiative heat transfer. Also, the actual resistance of the sensor should be calculated based on the temperature and the material properties of platinum, which is not included in this model.
