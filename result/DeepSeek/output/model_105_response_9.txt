
Use the Modelica model named `VanDerPol` to implement a Van der Pol oscillator with the parameter 'lambda = 0.3'. The system consists of two variables, 'x' and 'y', both starting at 1. The equations should define the derivatives of 'x' and 'y' to simulate the nonlinear oscillation behavior based on the Van der Pol relationship.

//modelica
```model VanDerPol
  parameter Real lambda = 0.3;
  Real x(start=1);
  Real y(start=1);
equation
  der(x) = y;
  der(y) = lambda*(1-x^2)*y - x;
end VanDerPol;
```

This model defines the Van der Pol oscillator with the given parameter 'lambda'. The equations for 'x' and 'y' are derived from the Van der Pol differential equations. The 'der' function is used to define the time derivative of 'x' and 'y'.

The initial conditions for 'x' and 'y' are set to 1. The simulation can be run from t=0 to t=100. The results can be plotted to visualize the behavior of the Van der Pol oscillator.

Note: This model does not include any control or input signals. It simply simulates the behavior of a Van der Pol oscillator with the given parameters.
