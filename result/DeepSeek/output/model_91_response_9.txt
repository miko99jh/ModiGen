
We have a container, and we want to keep the height of the water (`waterLevel`) in the container at a target height (`target_height`) of 1 meter. The water height (`waterLevel`) in the pool starts at 0.2m, and the pool will leak 0.1m per unit time. Use proportional integral control to implement the above control system (defining height error as `error`, and splitting the logic into proportional control `Pcontrol`, which handles the leak, and integral control `Icontrol`) and simulate the change of the water level over time, where the proportional coefficient (`Kp`) is 1 and the integral coefficient (`Ki`) is 0.5. Please help me implement a modelica model named `PIController`.

//modelica
```model PIController
  parameter Real waterLevel(start=0.2) "Current water level in the container";
  parameter Real target_height(start=1) "Target water level";
  parameter Real leakRate(start=0.1) "Rate at which water leaks out";
  parameter Real Kp(start=1) "Proportional coefficient";
  parameter Real Ki(start=0.5) "Integral coefficient";
  Real error "Height error";
  Real Pcontrol "Proportional control";
  Real Icontrol "Integral control";
  Real controlOutput "Control output";
initial equation
  error = target_height - waterLevel;
equation
  Pcontrol = Kp * error;
  Icontrol = Ki * integral(error, 0);
  controlOutput = Pcontrol + Icontrol;
  der(waterLevel) = -leakRate + controlOutput;
end PIController;
```

This model is a simple proportional integral controller that maintains the water level in the container at the target height. The proportional control term (`Pcontrol`) is proportional to the error between the current water level and the target height, and the integral control term (`Icontrol`) is proportional to the integral of the error over time. The control output is the sum of the proportional and integral control terms. The rate at which the water level changes (`der(waterLevel)`) is the sum of the leak rate and the control output.

Please note that this model assumes that the container has a constant volume and that the leak rate is constant. In a real-world scenario, these assumptions may not hold, and the model would need to be adjusted accordingly.

Also, please note that the integral term is calculated using the `integral` function, which calculates the definite integral of a function over a time interval. In this case, the time interval is from 0 to the current time (`time`). The `integral` function is used to calculate the integral of the error over time, which is then multiplied by the integral coefficient (`Ki`) to get the integral control term.

Finally, please note that the `der` function is used to calculate the derivative of a variable with respect to time. In this case, the derivative of the water level is the rate at which the water level is changing. The rate at which the water level is changing is the sum of the leak rate and the control output, which is the rate at which water is being added to the container.

This model can be used to simulate the change in the water level in the container over time. The water level will start at 0.2m and will change according to the control output until it reaches the target height of 1m. The control output will be the sum of the proportional and integral control terms, which will cause the water level to change until it reaches the target height.
