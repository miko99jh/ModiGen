
Generate a Modelica model named `Reactions_Array`for simulating chemical reactions using arrays. This model should accurately represent a chemical reaction dynamics by using arrays to handle multiple reaction components simultaneously. The model inputs include concentrations of chemicals as a real array 'C[3]', and reaction rate constants as a parameter array 'k[3]' with default values {0.1, 0.1, 10}. The model should initiate with an initial concentration array 'C = {1, 1, 0}'. The output should be the rate of change of concentrations, governed by the differential equations set in the 'equation' section. The value of 'der(C)' is determined by '{-k[1]*C[1]*C[2] + k[2]*C[3], -k[1]*C[1]*C[2] + k[2]*C[3] - k[3]*C[2]*C[3], k[1]*C[1]*C[2] -k [2]*C[3] -k [3]*C[2]*C[3]}'.

//modelica
```model Reactions_Array
  parameter Real C[3] = {1, 1, 0} "Concentrations of chemicals";
  parameter Real k[3] = {0.1, 0.1, 10} "Reaction rate constants";
  Real d_C[3] "Rate of change of concentrations";
equation
  der(C) = d_C;
  d_C = {-k[1]*C[1]*C[2] + k[2]*C[3], -k[1]*C[1]*C[2] + k[2]*C[3] - k[3]*C[2]*C[3], k[1]*C[1]*C[2] - k[2]*C[3] - k[3]*C[2]*C[3]};
end Reactions_Array;```

The model is named `Reactions_Array` and it has two parameters, `C` and `k`. `C` is an array of concentrations of chemicals, and `k` is an array of reaction rate constants. The model also has a state variable `d_C` which represents the rate of change of the concentrations. The equations in the model represent the rate of change of the concentrations as a function of the reaction rate constants and the current concentrations. The model is set to initiate with an initial concentration array `C = {1, 1, 0}`.

The model is a good example of how to use arrays in Modelica to represent multiple chemical reactions. It is also a good example of how to use differential equations to model chemical reaction dynamics.
