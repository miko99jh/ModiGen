The input is the current temperature (`T`) and the output is the resistance value (`Rt`) of the platinum resistor. It is known that the reference resistance at 0°C is `R0` = 100 ohms. The function should use an `algorithm` block and define `R0` along with constants `A` = 3.9083*10^(-3), `B` = -5.775*10^(-7), and `C` = -4.183*10^(-12) as protected internal variables. ```function Rt_T(Real T)
  input Real T "Current temperature";
  output Real Rt "Resistance value";
protected

  Real C = -4.183*10^(-12) "Constant";
algorithm

    Rt := R0;
  end if;
end Rt_T;```
