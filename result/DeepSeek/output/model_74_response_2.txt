
Generate a Modelica model named `Brush` that represents the voltage drop of carbon brushes in an induction machine. The model should include the following components and details:
1. Model Functionality: The model considers the voltage drop of carbon brushes and includes thermal effects.
2. Components:
- The model should extend from Modelica.Electrical.Polyphase.Interfaces.TwoPlug.
- Parameter for brush loss parameters: parameter Modelica.Electrical.Machines.Losses.BrushParameters brushParameters.
- The model should extend from Modelica.Thermal.HeatTransfer.Interfaces.PartialConditionalHeatPort with useHeatPort set to false and final T set to 293.15.
- Array of brushes: Modelica.Electrical.Machines.Losses.DCMachines.Brush brush[3] with each final brushParameters set to brushParameters and each final useHeatPort set to true.
3. Connections:
 The model should include the following connections:
- Electrical Circuit: Interpose the brush array between the polyphase plugs. Connect the internal pins of the positive plug ('plug_p.pin') to the positive pins of the brush array ('brush.p'), and route the negative pins of the brush array ('brush.n') to the internal pins of the negative plug ('plug_n.pin').
- Thermal Feedback: Implement a 'for' loop to collect the heat generated by each individual brush ('brush[j]'). Connect the heat port of every brush in the array ('brush[j].heatPort') to the model's inherited internal heat port ('internalHeatPort').

Please ensure that the model is syntactically correct and adheres to the Modelica language standard.

//modelica
```model Brush
  extends Modelica.Electrical.Polyphase.Interfaces.TwoPlug;
  parameter Modelica.Electrical.Machines.Losses.BrushParameters brushParameters;
  extends Modelica.Thermal.HeatTransfer.Interfaces.PartialConditionalHeatPort(useHeatPort=false, final T=293.15);
  Modelica.Electrical.Machines.Losses.DCMachines.Brush brush[3] annotation (final brushParameters = brushParameters, final useHeatPort = true);

equation 
  // Electrical Circuit
  plug_p.pin.n = brush.p;
  brush.n = plug_n.pin.p;

  // Thermal Feedback
  for i in 1:3 loop
    brush[i].heatPort.port.H = internalHeatPort.port.H;
  end for;
end Brush;
```

Please note that the brush model is a simplification and does not account for the spatial distribution of brushes on the stator, the effect of mechanical forces on the brushes, or the effect of the brushes on the air gap flux.
