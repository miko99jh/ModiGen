
Please use modelica to help me implement a model named `BouncingBall`: This is a bouncing ball model with a radius of 0.1m, an initial height of 1m and an initial velocity of 0 m/s. Affected by gravity, the ball moves in free fall and bounces upward at the moment it hits the ground. The recovery coefficient during the collision is 0.9.

//modelica
```model BouncingBall
  parameter Real r = 0.1 "Radius of the ball";
  parameter Real h0 = 1 "Initial height of the ball";
  parameter Real v0 = 0 "Initial velocity of the ball";
  parameter Real g = 9.81 "Acceleration due to gravity";
  parameter Real e = 0.9 "Recovery coefficient";
  Real h(start=h0) "Height of the ball";
  Real v(start=v0) "Velocity of the ball";
  Real a "Acceleration of the ball";
  Boolean bounce = false "Bounce flag";
equation
  // Newton's second law
  a = g - v*v/h*e;
  // Ball motion
  der(v) = a;
  der(h) = v;
  // Bounce condition
  if h < 0 and not bounce then
    v = -v*e;
    h = 0;
    bounce = true;
  elseif h >= 0 and bounce then
    bounce = false;
  end if;
end BouncingBall;
```

This model uses Newton's second law to calculate the acceleration of the ball, and then uses the equations of motion to calculate the velocity and height of the ball. The bounce condition is checked in each time step, and if the ball hits the ground, its velocity is reversed and the bounce flag is set to true. When the ball is above the ground, the bounce flag is set to false.
