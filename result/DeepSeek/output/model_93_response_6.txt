
We have a container, and we want to keep the height of the water (`waterLevel`) in the container at a target height (`target_height`) of 1 meter. The water height (`waterLevel`) in the pool starts at 0.2m, and the pool will leak 0.1m per unit time. Use PID control to implement the above control system (defining the height error as `error`, and splitting the logic into proportional control `Pcontrol` which handles the leak, integral control `Icontrol`, and differential control `Dcontrol`) and simulate the change of the water level in the pool over time with a proportional coefficient (`Kp`) of 1, an integral coefficient (`Ki`) of 0.2, and a differential coefficient (`Kd`) of 0.1. Please help me implement a modelica model named `PIDController`.

//modelica
```model PIDController
  parameter Real Kp = 1;
  parameter Real Ki = 0.2;
  parameter Real Kd = 0.1;
  Real error;
  Real Pcontrol;
  Real Icontrol;
  Real Dcontrol;
  Real waterLevel = 0.2;
  Real target_height = 1;
  Real leak = -0.1;
  Real control_output;
  initial equation 
    error = target_height - waterLevel;
    Pcontrol = Kp * error;
    Icontrol = Ki * error;
    Dcontrol = Kd * (error - pre(error));
    control_output = Pcontrol + Icontrol + Dcontrol;
  equation 
    der(waterLevel) = control_output + leak;
end PIDController;
```

This model is a very simple model of a PID controller. It calculates the error between the current water level and the target height, then calculates the proportional, integral, and derivative controls based on this error. The control output is the sum of these three controls. The water level is then updated based on this control output and the rate at which the water leaks from the pool.
