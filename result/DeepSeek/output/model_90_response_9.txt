
We have a reservoir, and we want to keep the height of the water (`waterLevel`) in the reservoir at a target height (`target_height`) of 1m. The water height (`waterLevel`) in the pool is 0.2m at the beginning. Please use proportional control to implement the above control system (defining the altitude error as `error` and the control signal as `control`) and simulate the change of the water level in the pool over time with a proportional factor (`Kp`) of 1. Please help me implement a modelica model named `PController`.

//modelica
```model PController
  parameter Real target_height = 1 "Target height of water level";
  parameter Real Kp = 1 "Proportional factor";
  Real waterLevel(start = 0.2) "Current water level in the pool";
  Real error "Error in water level";
  Real control "Control signal";
equation 
  error = target_height - waterLevel;
  control = Kp * error;
  der(waterLevel) = control;
end PController;
```

This model defines a simple proportional controller for a water level control system. The error is calculated as the difference between the target height and the current water level. The control signal is then calculated by multiplying the error by the proportional factor. The change in the water level is then determined by the control signal.

The `start` attribute is used to specify the initial value of the `waterLevel`. In this case, it is set to 0.2m. The `parameter` keyword is used to define `target_height` and `Kp`. These are constants that define the target water level and the proportional factor, respectively.

The `equation` block is used to define the equations that describe the system. In this case, there are three equations:

1. `error = target_height - waterLevel`: This equation calculates the error as the difference between the target height and the current water level.

2. `control = Kp * error`: This equation calculates the control signal as the proportional factor multiplied by the error.

3. `der(waterLevel) = control`: This equation determines the change in the water level as the control signal. The `der` function is used to calculate the derivative of `waterLevel`, which represents the rate of change of the water level.

Finally, the `end` keyword is used to close the model.

This model can be used to simulate the water level control system. The `target_height` and `Kp` parameters can be adjusted to change the target water level and the proportional factor, respectively. The `waterLevel` variable can be monitored to see how it changes over time in response to the control signal.
