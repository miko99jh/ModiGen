
We have a container, and we want to keep the height of the water (`waterLevel`) in the container at a target height (`target_height`) of 1 meter. The water height (`waterLevel`) in the pool starts at 0.2m, and the pool will leak 0.1m per unit time. Use PID control to implement the above control system (defining the height error as `error`, and splitting the logic into proportional control `Pcontrol` which handles the leak, integral control `Icontrol`, and differential control `Dcontrol`) and simulate the change of the water level in the pool over time with a proportional coefficient (`Kp`) of 1, an integral coefficient (`Ki`) of 0.2, and a differential coefficient (`Kd`) of 0.1. Please help me implement a modelica model named `PIDController`.

//modelica
```model PIDController
  parameter Real Kp=1 "Proportional gain";
  parameter Real Ki=0.2 "Integral gain";
  parameter Real Kd=0.1 "Derivative gain";
  parameter Real target_height=1 "Target height";
  parameter Real initial_waterLevel=0.2 "Initial water level";
  parameter Real leak_rate=-0.1 "Leak rate";
  Real waterLevel(start=initial_waterLevel) "Water level";
  Real error "Height error";
  Real Pcontrol "Proportional control";
  Real Icontrol "Integral control";
  Real Dcontrol "Differential control";
  Real integral(start=0) "Integral of error";
  Real derivative(start=0) "Derivative of error";
initial equation
  error = target_height - initial_waterLevel;
equation
  // Implement PID control here
  Pcontrol = Kp * error;
  integral = integral + Ki * error * timeStep;
  derivative = Kd * (error - derivative) / timeStep;
  Icontrol = integral;
  Dcontrol = derivative;
  waterLevel = Pcontrol + Icontrol + Dcontrol + leak_rate;
end PIDController;
```

The above model is a simple implementation of a PID controller. The proportional control (`Pcontrol`) is calculated by multiplying the error by the proportional gain (`Kp`). The integral control (`Icontrol`) is calculated by integrating the error over time, and then multiplying the result by the integral gain (`Ki`). The differential control (`Dcontrol`) is calculated by taking the derivative of the error, and then multiplying the result by the derivative gain (`Kd`). The total control output is the sum of the proportional, integral, and differential controls. The water level is then updated by adding the control output and the leak rate.

Also, the model assumes that the error is calculated as the difference between the target height and the current water level.
