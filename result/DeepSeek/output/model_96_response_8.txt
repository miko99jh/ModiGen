
Simulate the flow of fluid through a pipe. The known pressure at both ends of the pipeline are `P1` = 1080 and `P2` = 900. The pipe length (`L`) and inner diameter (`D`) are 1 and 0.05, respectively. The viscosity of the fluid (`mu`) is 2e-3. Please help me realize the Modelica model named `Pipe` corresponding to the above system to calculate a flow coefficient (`c`), the flow in the pipeline (`Q`), and the velocity (`v`). The model should use the Hagen-Poiseuille relationship where the flow (`Q`) is determined by the pressure difference and the coefficient (`c`), and the velocity (`v`) is derived from the flow and the pipe's cross-sectional area.

//modelica
```model Pipe
  parameter Real P1(unit="Pa") = 1080;
  parameter Real P2(unit="Pa") = 900;
  parameter Real L(unit="m") = 1;
  parameter Real D(unit="m") = 0.05;
  parameter Real mu(unit="Pa*s") = 2e-3;
  Real c(unit="m^4/(kg*s^2)");
  Real Q(unit="m^3/s");
  Real v(unit="m/s");
equation
  c = (8*mu*L)/(pi^2*D^4);
  Q = c*(P1 - P2);
  v = Q/(pi*D^2/4);
end Pipe;
```

Please note that the model assumes laminar flow and incompressible fluid, which are valid assumptions for this problem.
