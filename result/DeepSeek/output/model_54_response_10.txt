This model is part of the `Modelica.Electrical.Analog.Examples.OpAmps` package and extends the `Modelica.Icons.Example`.
The model should have the following parameters:
- `Vin` (Modelica.Units.SI.Voltage, default value: 5) representing the amplitude of the input voltage
- `f` (Modelica.Units.SI.Frequency, default value: 10) representing the frequency of the input voltage.

The model should include the following components:
- `ground` (Modelica.Electrical.Analog.Basic.Ground) to represent the ground connection.
- `vIn1` (Modelica.Electrical.Analog.Sources.SineVoltage) to generate a sinusoidal input voltage with parameters `V = Vin` and `f = f`.
- `vIn2` (Modelica.Electrical.Analog.Sources.ConstantVoltage) to generate a constant input voltage with parameter `V = Vin`.
- `vOut` (Modelica.Electrical.Analog.Sensors.VoltageSensor) to measure the output voltage.
- `feedback` (Modelica.Electrical.Analog.Examples.OpAmps.OpAmpCircuits.Feedback) to implement the feedback loop, with the initial current `i` of `p1_2` set to 0.

The following connections should be made:
- Dual-Source Integration: Route the dynamic signal 'vIn1' and the static signal 'vIn2' into the 'feedback' network's input ports 'p1' and 'p1_2' respectively.
- Reference & Return: Establish a common reference by tying the return pins ('.n') of both sources ('vIn1', 'vIn2') and the feedback input ('n1') to the 'ground.p'.
- Output Differential: Link the output stage of the feedback circuit ('p2', 'n2') to the 'vOut' sensor. Note: Ensure the polarity is set such that 'p2' connects to the sensor's negative terminal.

The model should accurately represent the behavior of an inverting subtracter circuit.

//modelica
```model Subtracter
  extends Modelica.Icons.Example;
  parameter Modelica.Units.SI.Voltage Vin=5 "Amplitude of the input voltage";
  parameter Modelica.Units.SI.Frequency f=10 "Frequency of the input voltage";
  Modelica.Electrical.Analog.Basic.Ground ground;
  Modelica.Electrical.Analog.Sources.SineVoltage vIn1(V=Vin, f=f);
  Modelica.Electrical.Analog.Sources.ConstantVoltage vIn2(V=Vin);
  Modelica.Electrical.Analog.Sensors.VoltageSensor vOut;
  Modelica.Electrical.Analog.Examples.OpAmps.OpAmpCircuits.Feedback feedback(initiali=0);
equation 
  connect(vIn1.p, feedback.p1) annotation(Line(color={0,0,128}, smooth=Smooth.None));
  connect(vIn2.p, feedback.p1_2) annotation(Line(color={0,0,128}, smooth=Smooth.None));
  connect(vIn1.n, vIn2.n) annotation(Line(color={0,0,128}, smooth=Smooth.None));
  connect(feedback.n1, ground.p) annotation(Line(color={0,0,128}, smooth=Smooth.None));
  connect(feedback.p2, vOut.plug_p) annotation(Line(color={0,0,128}, smooth=Smooth.None));
end Subtracter;
```

This is a simple example of a Modelica model for a DC motor. The model is based on the following equations:

- v = R * i + L * di/dt
- J * dω/dt = Kt * i - B * ω

where:
- v is the voltage applied to the motor,
- i is the current through the motor,
- R is the resistance of the motor,
- L is the inductance of the motor,
- ω is the angular velocity of the motor,
- J is the moment of inertia of the motor,
- Kt is the torque constant of the motor,
- B is the damping coefficient of the motor.

The model is a simple representation of a DC motor and does not include more complex dynamics such as friction or windage.

```modelica
model DCMotor
  parameter Real R(unit="V/A") "Motor resistance";
  parameter Real L(unit="H") "Motor inductance";
  parameter Real J(unit="kg*m^2") "Moment of inertia";
  parameter Real Kt(unit="N*m/A") "Torque constant";
  parameter Real B(unit="N*m*s/rad") "Damping coefficient";

  Real i(unit="A") "Current through the motor";
  Real v(unit="V") "Voltage applied to the motor";
  Real omega(unit="rad/s") "Angular velocity of the motor";

equation 
  // Voltage equation
  v = R * i + L * der(i);

  // Torque equation
  J * der(omega) = Kt * i - B * omega;
end DCMotor;
```

The following connections should be made:
- Connect the positive terminal of the `vIn` source to the positive terminal of the resistor.
- Connect the negative terminal of the `vIn` source to the `ground`.
- Connect the positive terminal of the resistor to the positive terminal of the `iOut` sensor.
- Connect the negative terminal of the resistor to the `ground`.

The model should accurately represent the behavior of a simple resistor circuit.

//modelica
```model Resistor
  extends Modelica.Icons.Example;
  parameter Modelica.Units.SI.Resistance R=1000 "Resistance of the resistor";
  Modelica.Electrical.Analog.Basic.Ground ground;
  Modelica.Electrical.Analog.Sources.SineVoltage vIn(V=10, f=100);
  Modelica.Electrical.Analog.Sensors.CurrentSensor iOut;
equation 
  vIn.p + R * iOut.p = 0 annotation(Line(color={0,0,128}, smooth=Smooth.None));
  vIn.n = ground.p annotation(Line(color={0,0,128}, smooth=Smooth.None));
  iOut.n = ground.p annotation(Line(color={0,0,128}, smooth=Smooth.None));
end Resistor;
```
