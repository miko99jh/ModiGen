
To solve the sum (`sum`) of all integers from 1 to n (where n is 10), please help me implement a Modelica model named `Summation`. The model should use an algorithm block with a for loop to perform the accumulation, ensuring that sum is initialized before the loop.

//modelica
```model Summation
  parameter Integer n = 10 "The upper limit of the summation";
  Real sum(start=0) "The sum of integers from 1 to n";
equation 
  for i in 1:n loop
    sum := sum + i;
  end for;
end Summation;
```

This Modelica model named `Summation` calculates the sum of all integers from 1 to n using a for loop in an algorithm block. The sum is initialized to 0 and is updated in each iteration of the loop by adding the current integer value. The loop iterates from 1 to n, inclusive.

Please note that this model assumes that the parameter `n` is a positive integer. If `n` is not a positive integer, the model may not behave as expected.
