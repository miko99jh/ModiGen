
We have a reservoir, and we want to keep the height of the water (`waterLevel`) in the reservoir at a target height (`target_height`) of 1m. The water height (`waterLevel`) in the pool is 0.2m at the beginning. Please use proportional control to implement the above control system (defining the altitude error as `error` and the control signal as `control`) and simulate the change of the water level in the pool over time with a proportional factor (`Kp`) of 1. Please help me implement a modelica model named `PController`.

//modelica
```model PController
  parameter Real target_height = 1 "Target height of the water";
  parameter Real Kp = 1 "Proportional control factor";
  Real waterLevel(start = 0.2) "Current height of the water";
  Real error "Error in height";
  Real control "Control signal";
equation
  error = target_height - waterLevel;
  control = Kp * error;
  der(waterLevel) = control;
end PController;
```

This model defines a proportional controller for maintaining a constant water level in a reservoir. The error is calculated as the difference between the target height and the current water level. The control signal is then calculated as the proportional factor times the error. The change in water level is then calculated as the control signal.

The `start` attribute is used to set the initial value of the `waterLevel` variable. The `der` function is used to calculate the derivative of the `waterLevel` variable with respect to time.

The proportional control factor `Kp` is a design parameter that determines the sensitivity of the controller to the error. A larger `Kp` value will result in a more sensitive controller, but it may also lead to oscillations or instability if not tuned properly.

The `target_height` parameter is the desired height of the water in the reservoir. The controller will strive to maintain the `waterLevel` at this height.

The `waterLevel`, `error`, and `control` variables are all continuous variables that can take on any real value. The `waterLevel` variable represents the current height of the water in the reservoir, the `error` variable represents the difference between the target height and the current water level, and the `control` variable represents the control signal output by the controller.

The `der` function is used to calculate the derivative of the `waterLevel` variable with respect to time. This derivative is equal to the control signal, which means that the change in water level over time is determined by the control signal.

The `start` attribute is used to set the initial value of the `waterLevel` variable. In this case, the initial water level is set to 0.2m.

The `equation` block is used to define the equations that govern the behavior of the model. In this case, there are three equations: one for calculating the error, one for calculating the control signal, and one for calculating the derivative of the water level.

The `end PController` statement marks the end of the model.
