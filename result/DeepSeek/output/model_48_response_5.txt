
Generate a Modelica model named `Inductor` that represents an ideal linear electrical inductor. The model should extend the 'Modelica.Electrical.Analog.Interfaces.OnePort' interface which provides a single electrical port with current and voltage. The initial value of the current 'i' in it is set to 0 in extends. The main parameter of the model is 'L', representing inductance with the SI unit 'Modelica.Units.SI.Inductance'. Ensure that the initial value of the inductance 'L' is set to 1 henry. The model should include an equation that relates the inductance to the time derivative of the current (i.e., L * di/dt = v, where 'i' is the current and 'v' is the voltage).

//modelica
```model Inductor
  extends Modelica.Electrical.Analog.Interfaces.OnePort;
  parameter Modelica.Units.SI.Inductance L=1 "Inductance";
  Modelica.SIunits.Current i_start(start=0) "Initial current";
equation 
  v = L * der(i);
end Inductor;
```

In this model, the voltage 'v' is defined as the product of the inductance 'L' and the time derivative of the current 'i'. This equation represents the basic law of electromagnetism that states the voltage across an inductor is equal to the rate of change of the current through it.
